# 契灵术 · 契约类型

> 五种契灵，五种契约。每种都是一条不同的道路。
> 选残响的人想继承历史。选地缚灵的人想扎根。选概念体的人想理解不可理解之物。选镜中我的人——在寻找自己。选天道碎片的人不要命。

---

## 一、残响（Echoes）——逝者的遗产

### 本质

残响是已故修仙者在织上留下的意志碎片。当一位修士陨落时，其意识的大部分消散回归本源，但某些强烈的情感、执念或技艺会在织上留下一个"印痕"。这些印痕在特定条件下会被唤醒，形成一种半意识的灵性存在。

残响不是灵魂，不是鬼魂。它们是**被剥离了自我的记忆束**。它们没有完整的意志，只有残留的冲动和习惯。

### 给予的能力

残响通常继承自生前修士的能力——虽然打了折扣：

1. **技艺传承**：获得死者生前的某项技能（如剑法直觉、炼器判断力、丹方记忆）。继承者不需要练习就能使用，但精度和深度不及原主——大概六到八成。
2. **战斗本能**：获得死者的战斗经验和肌肉记忆。签约者会发现自己"知道"在某一情境下该怎么做，但不是自己学会的。
3. **感知残响**：能感知到附近是否存在类似残响或灵体——这是残响类契约最常见的附加能力。
4. **记忆回响**：偶尔在梦中或冥想中体验死者生前的人生片段。不是能力——是副作用，但有些契主学会了从中提取信息。

### 代价

残响的典型代价是**记忆**（见 [03.全新异能体系/04_代价与反噬.md#记忆](/03.全新异能体系/04_代价与反噬.md#记忆)）：

- 轻微：遗忘近期琐事（昨天吃了什么、上周见过谁）
- 中等：遗忘重要信息（某人的脸、自己的生日、某个重要约定）
- 严重：遗忘身份记忆（自己的名字、家人、重要人生经历）
- 极端：遗忘自我——不知道"自己是谁"仅留下躯壳

代价的本质是**身份侵蚀**：每从残响那里获得一份过去的记忆，就需要付出自己的一份现在记忆作为交换。残响没有自己的记忆可支付——它就用你的。

### 如何找到残响

- 前往已知陨落地点，在死者陨落的精确时间（精确到时辰）以共鸣法阵呼唤。
- 需要知道死者的真名和至少一项显著人生经历作为"关键词"。
- 越强大的修士留下的残响越清晰，但也越危险——化神以上修士的残响可能残留部分意识，签约难度极高。

### 危险信号

- 契约后开始不自觉使用签约者生前习惯（语气、手势、偏好）
- 出现不属于自己的记忆——而且是连续的、有叙事结构的（说明残响正在尝试"覆盖"你的记忆）
- 梦见自己以第一视角经历签约者生前的死亡场景（反复出现）

### 案例：断剑陈

一个名为陈不鸣的金丹期剑修在挑战化神期修士时被一剑斩杀。他的残响被一个名叫阿七的散修捕捉签约。

阿七获得了陈不鸣的剑意直觉——他从此"天生就会用剑"。

代价是阿七开始遗忘记忆。最初是剑招的口诀（反正他已经有了剑意直觉，不需要口诀了）。然后是教他剑招的师父的脸。然后是师父的名字。签约两年后，阿七想起"师父"这个词时有情感反应但想不起任何具体的人。

阿七仍然是一名优秀的剑客。他不知道自己为什么用剑。他只是用。

## 二、地缚灵（Bound Spirits）——地点的记忆

### 本质

地缚灵是**被绑定在特定地理位置**的灵性意识。它们通常由"某个地点反复发生的事件类型"在织上留下的印记凝结而成。一口被遗弃了数百年的枯井、一座战争废墟、一间无人问津的老图书馆、一条发生过重大交通事故的路口——都可能孕育地缚灵。

地缚灵不是"鬼魂占房子"。地缚灵就是那个地点本身——或者说，**那个地点在织上的"指纹"产生了意识**。

### 给予的能力

地缚灵的契约能力与它的绑定地点高度相关：

1. **环境感知**：能感知绑定地周围一定范围内的所有生命活动和能量波动。感知半径与契约深度成正比。
2. **领域微调**：在绑定地附近，可以对环境进行轻微的调整——改变温度、湿度、光线走向、声音传播方向。越强大的地缚灵允许的调整幅度越大。
3. **主场优势**：在绑定地范围内，契主的其他契约能力获得稳定性加成（锚点借用地理锚点）。
4. **地点记忆提取**：高级契约可以从地缚灵那里读取该地点历史上发生过的重大事件（地缚灵的"记忆"）。
5. **短暂形态借用**：极深契约下，契主可以短暂借用该地点的物质形态（例如调用古井的井水形成攻击、催动废墟的石块形成防御）。代价极高。

### 代价

地缚灵的典型代价是**自由**（见 [03.全新异能体系/04_代价与反噬.md#自由](/03.全新异能体系/04_代价与反噬.md#自由)）：

- 轻微：离开绑定地超过一定时间会感到不适（焦虑、失眠）
- 中等：离开绑定地后能力大幅减弱，身体出现轻度病症
- 严重：无法离开绑定地超过24小时，否则契约自动进入违约状态
- 极端：完全绑定在特定区域，离开即死（少数极端案例）

代价的本质是**领地化**：你占据了地缚灵的领地，所以你也必须成为这片领地的一部分。地缚灵没有可移动性，于是它用契约把你的可移动性也锁在它身边。

### 如何找到地缚灵

- 搜索有"异常历史"的地点——通常是经历了长期、强烈、重复的情感事件的地方
- 需要在该地点持续停留至少3-7天，以特定频率敲击织（通过呼吸法、冥想或小型法阵）
- 地缚灵回应时通常通过环境变化传达（温度骤变、奇特的回声、异常的风向）

### 危险信号

- 对该地点产生"思乡病"级别的依恋——离开时出现生理性戒断反应
- 开始认为其他地点"不安全""不对""不是家"
- 在绑定地之外出现现实感丧失（depersonalization）——感觉自己是"不完整的"
- 持续梦见该地点——但梦中的地点与真实地点有细节差异（说明地缚灵正在扩张或改变你对它的认知）

### 案例：枯井翁

枯井翁是[契约者丙](/03.全新异能体系/01_体系起源.md#镜的第一次直接观察)案例中提到的地缚灵。它绑定在一口位于荒村中的古井里。这口井的历史可以追溯到五百年前——据说在饥荒年间，全村人靠这口井活了下来，但有十几个走不动的老人跳井自杀，为年轻人省下口粮。

枯井翁虽然名为"翁"，但它没有"想当老人"的意识。它是一个**生存与牺牲的混合体**——给予生存资源，索要陪伴和忠诚。它的契约在早期记录中非常常见，但随着该井所在村庄彻底荒废，近年已无人与它续约。枯井翁目前处于未签约的休眠状态。

## 三、概念体（Concepts）——抽象的有形化

### 本质

概念体是宇宙中最奇特的存在。它们是抽象概念在织上积累到足够密度后**自发产生的半意识体**。

当一个概念被人类社会反复使用、讨论、传承，它在织上的痕迹就会不断加深。"遗忘"不是一件东西——但被遗忘的事物在织上留下了大量的空白节点；"边界"不是一种物质——但每一道边界（国界、心墙、定义边界）都在织上叠加了纹路。当这些纹路密集到一定程度，它们有可能聚合为一个自我指涉的节点——这就是概念体的诞生。

概念体是**最小意识的节点**。它们能感知、能回应、能谈判——但它们对"人类"的理解天然有限。你无法向一个概念体解释"为什么你会笑"——因为"笑"是一个概念体，而不是一个有情感体验的存在。

### 给予的能力

1. **概念感知**：能感知到相关概念在本地区的"浓度"和活动。例如，与「遗忘」签约者能感知到周围哪些地点正在被遗忘、哪些记忆正在消退。
2. **概念干预**：在相关概念上施加微调。与「边界」签约——可以暂时软化或硬化一道边界（物理或抽象边界）。与「回声」签约——可以操控声音的延迟和重复。
3. **概念视野**：短暂切换到以该概念为核心的感知模式。例如「遗忘」视角下，世界是按"被记住程度"着色的——颜色越淡的事物越接近被遗忘。
4. **概念保护**：在相关概念的"攻击"面前获得抵抗力。与「沉默」签约者不会被声音类法术伤害。
5. **概念创造**：极深契约下，可以临时创造该概念的一个小规模实例——比如与「回声」签约者在死寂之地创造一道回声（需要极高的契约深度和锚点质量）。

### 代价

概念体的典型代价是**情感/感知**（见 [03.全新异能体系/04_代价与反噬.md#情感](/03.全新异能体系/04_代价与反噬.md#情感)）：

代价的形式非常抽象：**你给予概念体"人类的体验样本"**。

- 与「遗忘」签约：你逐渐失去"难以忘怀"的能力——重要记忆和琐碎记忆的权重差异减小，一切都变得可被遗忘。
- 与「沉默」签约：你逐渐失去对沉默的不适感——这听起来不错，但意味着你也失去了对危险寂静的警觉。
- 与「边界」签约：你逐渐模糊自己的心理边界——更容易共情，也更容易被他人情绪淹没。

这不是概念体"拿走"你的东西——是概念体**不理解为什么这些边界对你重要**。在概念体的逻辑中，更加"像概念"是一种优化，而非损失。

### 如何找到概念体

- 概念体不会主动寻找签约者——你需要自己"进入概念"。
- 前往该概念密度最高的环境：遗忘→废弃档案馆；沉默→地下洞穴深处；边界→国境线、门槛、桥梁。
- 在该环境下进行长时间的冥想，放下自我边界，尝试"感受概念本身"而不是感受环境。
- 概念体的回应非常微弱——大多数人在到达真正接触之前就已经放弃了。

### 危险信号

- 开始用概念的逻辑思考日常问题——如与「边界」签约者开始认为"人际关系是应该被软化的边界"
- 对概念的敏感性下降——需要更强烈的概念刺激才能感知到它的存在（说明你在被同化）
- 出现"概念漂移"——在签约后对与概念相关的话题产生极端观点或情绪钝化
- 其他人说你和签约前"不一样了"，但你自己无法感知到差异

### 案例：沉默的守门人

一个人称"老默"的契主与概念体「沉默」签订了契约。老默原本是一个普通的小镇管理员，负责看守一座废弃的战争纪念馆。

契约内容：沉默给予老默"在任何噪声环境中维持内心寂静"的能力。沉默索要：老默必须在有生之年每年累积至少30天完全不说话。

十年后，老默可以在一场摇滚演唱会中心如止水地看书。但他的表达欲望大幅降低——他发现自己越来越不愿意开口说话，即使在需要社交的场合。他的妻子离开了他，原因不是不爱了，是"你不再和我说话了"。

老默认为这笔交易是值得的。他喜欢内心的寂静。但镜的记录中有一条老默的体检异常：他的声带功能正常，但语言区域的大脑活跃度比签约前降低了37%。代价不仅是行为层面的——它已经写入了他的神经结构。

## 四、镜中我（Mirror Self）——平行自我的投影

### 本质

镜中我——其他体系有时称其为暗我、影我、对位自我——是契主在织的镜像层中的一个**平行版本**。

镜的存在是这一契约类型的先决条件——镜的观测行为在织上创造了一层"反射面"。在这层反射面上，每个被镜记录过的存在都会产生一个投影。这个投影不是灵魂、不是复制品——它是**织记录的关于"你"的全部信息的总和**，以某种仿佛有意识的方式呈现出来。

镜中我会说话、会回应、会讨价还价——但它不是你。它知道关于你的一切（因为它的构成材料就是你被记录的所有数据），但它不是你的视角。它可以预测你的反应，但不一定认同你的选择。

与镜中我签订契约，本质上是在与**自己的数据投影**做交易。

### 给予的能力

1. **自我认知**：镜中我能告诉你织上记录的关于你的所有数据——包括你自己不知道的。包括你遗忘的承诺、你潜意识中的偏好、你的行为模式中你自己看不到的盲点。
2. **真知之眼**：镜中我可以在一个特定时刻揭示"真实情况"——不是预测未来，而是在当下被多重信息包围时，告诉你哪一条是真的。一天限用一次，使用时消耗极大精神。
3. **镜像步**：以镜中我在镜像层的移动方式在物质界进行短距离位移。不是传送——是一种绕过中间路径的"跳跃"，看起来像闪烁。
4. **自我整合**：深契约下，可以暂时借调镜中我在镜像层的能力（镜中我理论上比你更"纯粹"——因为它只有数据，没有实体）。效果：短时间内的状态全满、感知全开——但结束后会经历剧烈崩溃期。

### 代价

镜中我的代价是独一无二的：**你必须面对你不愿面对的东西。**

这不是抽象的——在每次履约时，镜中我会揭示一个被你忽略或压制的事实。这个事实可能关于你自己、关于你身边的人、关于你正在做的事。每次揭示都相当于一次心灵冲击。

不支付代价发生的方式：你拒绝面对揭示的事实，或你逃避履约场景。后果：镜中我开始"裂开"——它在镜像层的存在变得不稳定，而你的自我认知也开始混乱。你开始不清楚哪些是你的真实记忆，哪些是镜像层的数据。

### 如何找到镜中我

- 必须通过镜。没有镜的协助，无法进入镜像层。这不是技术限制——是结构限制：镜不在场时，织的反射面不会稳定到足以支持签约。
- 签约场景需要在镜的见证下进行，在镜的"视野"中完成仪式。

### 危险信号

- 开始质疑自己所有决定的真实性——"是我选的还是镜中我选的？"
- 对镜产生过度依赖或强烈的敌意（两种都是危险信号——说明你无法区分自己和镜像层数据）
- 镜中我在谈判中引用了"你自己说过但你不记得的话"——且这句话是错误的（说明镜像层数据被污染）
- 出现镜像幻觉——在日常生活中看到自己的镜像做出与你不同的动作

### 案例：尤小姐

尤小姐是一位30出头的家族企业继承人。她精明、能干、社交手腕高超。但她总觉得自己"不知道自己在做什么"——她的成功似乎全是运气和本能，她无法理解自己的决策过程。

她通过镜的协助与镜中我签订了契约。仪式中她第一次正面看到了"织上的自己"——一个比她看起来更疲惫的女人。

镜中我揭示了第一条事实：尤小姐的决策模式是基于恐惧而非勇气——她所有的"成功选择"本质上都是在逃避她认为更糟的选项。她不是善于选择，而是善于识别灾难并远离它们。

这个事实让尤小姐崩溃了三周。之后她承认了这一点，接受了"自己的成功建立在恐惧之上"这个事实——并在此基础上开始了有意识的重建。

代价是持续的：每次履约，镜中我会告诉她又一件她不想知道的事。尤小姐说，与镜中我的契约是"我做过的最痛苦也是最有用的事"。

## 五、天道碎片（Heavenly Shards）——破碎的法则

### 本质

[01.系统/世界观底层规则.md#宇宙法则](/01.系统/世界观底层规则.md#宇宙法则)中提到的五条宇宙法则是不可违背的。但法则本身经历了漫长的宇宙演化——某些法则在某些事件中"碎裂"了，产生了碎片。这些碎片携带着法则的一部分力量，漂浮在织的深处。

天道碎片是宇宙演化过程中自然产生的"法则残余"。它们不是法则本身——它们是法则上脱落的小片。就像一面镜子碎了，碎片仍然能反射东西——但反射的内容是不完整的、畸变的。

每一片天道碎片都代表着一条宇宙法则的局部、不完整、扭曲的副本。

已知的碎片类型极为稀少——镜的记录中目前确认存在的不到二十片。以下为已知类别：

| 碎片 | 源于法则 | 能力 |
|:----|:--------|:-----|
| 代价豁免之隙 | 万物皆有其代价 | 在极小范围内"暂缓"一次代价的支付（不是免除，是延期） |
| 观测裂隙 | 观测影响现实 | 在一次事件中让自己不被任何观测手段锁定 |
| 不衡之衡 | 宇宙趋向平衡 | 在局部制造短时间的不平衡（让对手的"运气"被压制） |
| 深度借用 | 无体系可兼得广深 | 临时借用另一个体系的能力一次（但以自身体系能力降低为代价） |
| 因果断点 | 因果链 | 切断一条短因果链（让一件事不因其前因而发生） |

### 给予的能力

1. **法则微调**：在碎片对应的法则范围内进行小规模调整。效果视碎片而定——代价豁免之隙可以让一次代价支付延迟最多24小时。
2. **法则感知**：感知附近是否存在法则层面的异常——其他碎片、严重违逆法则的事件、或天道本身对某事物的注意。
3. **临时法则替换**：极高深度下，可以用碎片的逻辑临时替换局部区域的法则——代价豁免之隙可以让一整个区域在10秒内"代价豁免"（但结束后所有暂缓的代价一次性到账，加收20%滞纳金）。

### 代价

天道碎片索要的代价是**命理**（寿命、因果走向）和**天道的注意**：

- 每次使用碎片能力，都会在织上产生一次"异常波动"。这波动会引起天道的"注意"——不是人格化的注意，是法则层面的反馈。就像往平静的水面扔一块石头——天道会感知到涟漪。
- 天道注意的后果不可预测。可能是好运突然中断，可能是附近有人渡劫时劫雷威力翻倍，可能是什么都不发生——也可能是什么都发生。
- 命理代价：缩短寿命（以月或年计，视使用强度而定），或在因果链上制造一个小型"短路"——一件本来不该发生的事发生了，或一件本该发生的事没有发生。

### 如何找到天道碎片

**无法主动寻找。** 天道碎片随机出现在织密度异常的区域。没有规律可循。已知的寻找方式：

- 天道碎片探测器——一种需要耗费大量资源运行的法阵，能在碎片靠近时发出信号。有效半径约五十里。
- 通过深度解析某些古老秘籍中隐藏的织纹，可能定位碎片的可能出没区域——但每本秘籍只能使用一次，且成功率约15%。
- 等待碎片主动接触——极少数案例中碎片因为找不到合适的宿主而主动连接了一个偶然经过的敏感者。

### 危险信号

- 契约能力开始"溢出"合同约定——能做的比合同写得多（法则层面的契约泄漏）
- 身边出现无法解释的微小异常：物品消失又出现、时间感知紊乱、因果错位
- 天劫相关的自然现象增多——即使不是在渡劫期（说明天道在"检查"该区域）
- 其他契灵开始回避你——它们能感知到天道碎片在你身上的"气味"

### 案例：碎片持有者零号

代号"碎片零"是已知最早的天道碎片持有者。他在一次狩猎残响的途中偶然被一片"代价豁免之隙"碎片主动连接。

碎片零原本是个谨慎的人。他试图用碎片暂缓代价支付来"优化"他的契约组合。第一次使用：完美。第二次使用：完美。第三次使用：代价被暂缓了，但三天后他的房子在雷暴中被击中——整个街区只有他那一栋。

他没有死，但全身偏瘫。碎片从他身上脱离，寻找下一位宿主。

碎片零最后的事迹是他用最后一只还能动的手写下的警告：**"不要以为延迟支付等于减少支付。天道记账从不打折。"**
