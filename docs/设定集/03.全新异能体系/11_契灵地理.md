# 契灵术 · 契灵地理

> 修仙者看世界——灵脉如血管，灵气如血液，洞天福地如心脏。
> 契主看同一个世界——织如账簿，回响如脚注，裂缝如未结清的账目。
> 同一张地图，两种读法。两者都是对的。

---

## 一、契灵视角的世界观

### 1.1 织密度（Fabric Density）

[织](/01.系统/世界观底层规则.md#本源织) 并非均匀分布。它在世界不同区域有不同的密度——修仙者将其感知为"灵气浓度"，契主将其感知为"织密度"和"回响清晰度"。两个描述指向同一个物理现实，但测量的是不同属性。

**织密度等级**（契灵术社区的常用分级）：

| 等级 | 名称 | 表现 | 类比 |
|:----|:-----|:-----|:-----|
| 5 | 织溢 | 织物肉眼可见地扭曲光线；凡人进入会感到强烈压迫感甚至幻觉 | 账簿被墨水浸透 |
| 4 | 高密 | 回响信号清晰，无需仪式即可感知契灵存在；锚点建立极为稳固 | 账簿条目拥挤 |
| 3 | 适中 | 正常区域；签约需要仪式辅助，锚点维护标准成本 | 正常账簿 |
| 2 | 低密 | 回响信号微弱；契灵活动稀少；锚点维护需要额外精力 | 账簿页面稀疏 |
| 1 | 织稀 | 几乎无回响信号；契灵无法长期驻留；锚点易松动 | 只有账本封面 |
| 0 | 绝织 | 织密度趋零；任何契约行为均不可能；也是修仙的绝灵之地 | 没有账簿 |

**关键规律**：织密度每降低一级，锚点稳定性下降约40%，能力输出降低约30%，代价支付效率降低约25%——这意味着在低密度区签约需要支付更多代价才能获得同等能力。

### 1.2 回响强度（Echo Intensity）

回响强度不是织密度的同义词。高密度区不一定有高回响——如果那片区域历史上"平静"，织虽厚但不含印记。低密度区也可以有局部高回响——如果某个重大事件在薄织上留下了深刻划痕。

**回响强度的四种来源**：

1. **历史事件沉积**——战争、灾难、大规模死亡在织上留下的集体印记。残响的主要来源。
2. **概念长期积累**——某个概念被反复实践和讨论的区域，概念体容易在此凝聚。如法院附近凝聚"公正"概念。
3. **反复仪式痕迹**——同一地点被多次用于契约仪式，残留的锚点痕迹会增强该区域的回响敏感度。
4. **天道碎片沉降**——碎片在某区域停留时间越长，该区域的织越"规则化"——变得可预测，但也变得僵硬。

### 1.3 织裂谷（Fabric Rift）

织裂谷是织的结构性损伤区域——近似于布匹上的撕裂。裂谷处织不连续，深层织暴露于物质界。

**特征**：
- 裂谷本身不可见（不具备物理形态），但可通过以下现象识别：时间流速异常、镜子无法反射观察者、声音传播路径弯曲。
- 裂谷中的契灵异常活跃——它们可以"穿过裂缝"从深层织直接进入物质界，绕过正常的契约锚点结构。这意味着不需要签约就可以与契灵交互——极危险，也极诱人。
- 裂谷边缘的织密度极不稳定——有时高达等级5，有时骤降至等级1。无法预测的波动使这里成为不安全的签约地点。
- 修仙者感知织裂谷为"灵气混乱区"——修炼效率极低但偶尔出现稀有的灵气结晶。

**已知织裂谷**：目前确认的织裂谷有三处——一处位于 镜渊 的推测位置，一处位于某片深海沟壑中（未充分探索），一处位于 地缚林海 的中央地带。

## 二、契灵活跃区（Entity-Dense Regions）

以下六个区域在织上具有特殊的回响结构，是不同类型契灵的高活性地带。

### 1. 残响平原（Echo Plains）

**位置**：大陆中部偏西，一片看似普通的开阔平原。植被低矮，风力恒定。表面没有任何异常，但风声中有声音——如果仔细听。

**织特征**：织密度等级3（适中），但回响强度极高。这里是[上古纪和近古纪](/03.全新异能体系/01_体系起源.md)多次大规模战争的战场。每次战斗都在织上留下了深深的划痕——死者的意志碎片、法术余波、绝望和愤怒的印记——层层叠叠地沉积在这片区域的深层织中。

**活跃契灵类型**：残响（Echoes）——这里是残响密度最高的已知区域。镜记录了至少三十个独立的残响信号源，但其中大多数过于碎片化，无法进行有意义的契约谈判。

**安全等级**：中低。残响本身不主动攻击——它们只是"存在"。但长期暴露在密集的残响信号中，契主会经历身份侵蚀加速。普通人在此区域停留超过一周可能出现"不属于自己的记忆闪回"。修炼者则可能被残响中的战斗本能感染，变得易怒或好斗。

**对修仙者的影响**：灵气密度正常，但修炼效率下降约15%——因为浅层织被残响的振动干扰。神识灵根修炼者感受最明显。

**对契约者的价值**：
- 寻找残响签约的最佳地点——特别是那些追求战斗传承的契主（如[阿七](/03.全新异能体系/08_代表人物.md#一、阿七·执契者)这样的剑意继承者）。
- 残响中偶尔保留有陨落修士的功法记忆碎片——对研究古代修炼技术有价值的考古材料。
- 一些残响在消散前会"吐出"它们储存的代价（记忆束）——这是一种稀有但不稳定的资源。

**镜的备注**：残响平原不是一个地点——它是一个事件集。死在平原上的人比活着的更多。织上刻着的不是土壤，是遗言。

### 2. 地缚林海（Bound Spirit Forest）

**位置**：大陆东南部，一片绵延数千里的古老森林。树冠遮天蔽日，地面常年不见阳光。林中有明显的路径痕迹——但地图上标注的道路和林中实际的道路总是不一致。

**织特征**：织密度等级4（高密）。森林中的每一棵树都与一个地缚灵存在松散的绑定关系——不是正式的契约，而是"共生"：树木提供稳定的物理锚点，地缚灵提供对环境的微调能力（调节光照、土壤湿度、虫害水平）。

**活跃契灵类型**：地缚灵（Bound Spirits）——这里是地缚灵的最大聚集地。但与外界不同的是，林海中的地缚灵大多绑定于"树木"而非"建筑"或"地点"。树倒地缚灵移——这使它们具有有限的可移动性，在整个契灵体系中是罕见的。

**森林的心脏**：林海中央生长着一棵被称为 "母树" 的巨型古木——树龄超过八千年，树冠覆盖面积相当于一座小镇。母树本身就是一个未签约的地缚灵——它可能是整个地缚林海的第一棵树，所有其他地缚灵都是从它的织上分枝出去的。

- 母树是否有独立意志？存在争议。它从未对人类做出过回应，但它能"协调"整个林海的地缚灵活动。曾有契主试图在母树下进行沉默静坐仪式——七天后被发现时已陷入深度昏迷，身体无伤，但意识中少了七年的记忆。
- 母树不接受契约。至少——没有已知的成功案例。尝试过的人都没有死，但也没有成功。

**安全等级**：中。地缚灵不主动敌对——但森林本身会"引导"访客的行走路线。如果不按照森林"愿意"的方向走，可能会在同一条路上反复绕圈。这就是当地猎人所说的"被树留宿"。

**对修仙者的影响**：高织密度对修仙者来说是双刃剑——灵气充沛（修炼效率+20%），但灵根与地缚灵的共生关系产生干扰，长时间修炼后可能出现"灵根偏好偏移"（灵根倾向于与当地地缚灵产生不必要的共鸣）。

**对契约者的价值**：
- 寻找地缚灵签约的最佳地点。特别是那些追求"扎根"型契约的契主。
- 林海中的某些稀有植物是高级仪式材料——如"影苔"（只在母树根部生长，用于高级锚点加固仪式）。
- 一些古老的树木上刻有古代契主留下的契约记录——树皮愈合后形成的纹路记录了当时的织状态。

**镜的备注**：地缚林海不是森林在地面上生长——它在织上生长。每棵树都是一根锚点。走在这片森林里的人，其实是在一个巨大的锚点网络中穿行。

### 3. 概念荒漠（Concept Wasteland）

**位置**：大陆西部，一片辽阔的荒原。地表多为碎石和沙土，植被稀疏。白天酷热，夜晚严寒。这片区域在地图上的边界在不断变化——不是因为地理变迁，是因为"边界"本身在这里不稳定。

**织特征**：织密度等级1-2（低密到极低密），但具有异常的"概念浓度"。这里的织虽然薄，但被抽象概念密集地"覆盖"——就像一个磨损的桌面上有厚厚一层灰尘，灰尘不是桌面的一部分，但它覆盖了桌面。

**为什么概念在这里"可见"**：理论上，织密度越低，概念越难凝聚。但概念荒漠是一个反例——镜的理论是：这片区域在远古时期经历了多次织密度崩溃和重建，每次崩溃都在织上留下了"概念断层"。这些断层使抽象概念得以摆脱织的物理约束，呈现出半独立的存在状态。

**活跃契灵类型**：概念体（Concepts）——特别是与"极限"相关的概念体：干渴、迷失、终点、虚空、持久。这些概念体不像城市中的概念体（"沉默"、"回声"、"边界"）那样温和——它们是极端环境孕育的极端存在。

**异常现象**：
- **距离失效**：直线上目测一公里的距离，实际行走可能需要一小时或十分钟——两者之间的比例变化没有规律。
- **时间跳跃**：旅行者报告"丢失了几个小时"——不是昏迷，是意识连续但时间跳过了。一个人看着太阳落山，下一个瞬间太阳已经在同一位置升起——他丢失了整个夜晚。
- **身份模糊**：长期停留（超过三天）的访客开始不确定"我是谁"——不是记忆丧失，是自我感变得不扎实，像铅笔写的字被橡皮擦过。
- **因果松动**：因果关系变得可选而非必然。"我捡起一块石头"不一定导致"石头在我手里"——石头可能被捡起后出现在三米外。

**安全等级**：极高。概念荒漠是已知最危险的契灵活跃区之一。不建议非契主进入。不建议多契者单独进入。不建议任何人停留超过48小时。

**对修仙者的影响**：绝灵之地的变体——灵气极度稀薄，修炼不可能。体魄灵根持续消耗（身体需要更多能量维持基本功能）。神识灵根受到严重干扰（概念荒漠的"概念覆盖"对神识的干扰类似持续刮擦黑板的声音）。

**对契约者的价值**：
- 与极端概念体签约的唯一地点。"干渴"概念体只存在于真正缺乏水分的环境中。
- 概念荒漠中的生存本身就是一种资格证明——能在这里存活并维持契约的契主，在任何社区都会受到尊重。
- 高回报：这里的极端概念体提供的是其他地区无法获得的能力。人类——包括高境界修士——面对"干渴"时都是脆弱的。

**镜的备注**：概念荒漠在嘲笑"现实"这个词。它提醒所有体系——织上的一切都是模式，不是物质。荒漠只是摊开了底层代码让你看到。

### 4. 镜渊（Mirror Depths）

**位置**：不确定。理论上的坐标指向大陆东北部一座深不见底的天然竖井——当地人称 "无底渊"。竖井直径约三十米，向下垂直延伸，已知深度超过两千米仍未触底。井壁光滑，由一种黑色石材构成，不反射任何光线。但如果用镜子反射井口的光线照向井内——镜中会显示出一个与物理井完全不同的景象。

**织特征**：织密度等级不可测量——因为镜渊的织结构与其他任何区域都不一致。这里的织似乎"折叠"了——表层织和深层织之间的距离比正常区域短得多，因此镜像层与物质界几乎重叠。

**本质**：镜渊是 [镜](/01.系统/世界观底层规则.md#三、镜的角色) 的"背面"——是织上反射层的物理薄弱点。镜的观测行为在织上创造了一层反射面，而镜渊被推测为该反射面在物质界最接近的地方。如果把织比作一面巨大的镜子，镜渊就是这面镜子的"边缘"。

**活跃契灵类型**：镜中我（Mirror Self）——这里是所有镜中我契灵的"发源地"。不，更准确地说：镜中我并非从镜渊"出生"，但镜渊是与镜像层交互最直接的物理接口。

- 已知多位与镜中我签约的契主（如[尤小姐](/03.全新异能体系/08_代表人物.md#四、尤小姐·契主（4份契约）)）报告过同一个梦境：站在无底渊的边缘向下看，看到自己的倒影在井底向上看——但倒影更高、或更矮、或年龄不同。
- 镜自身对镜渊的存在不作评论。当被问及时，镜的标准回应是："记录显示该位置存在。镜无法确认其性质。"

**安全等级**：极高。不是因为物理危险（虽然摔下去肯定死），而是因为存在性危险。接近镜渊的人报告了严重的现实感丧失——他们开始不确定自己是"原版"还是"镜像"。有记录显示，一个在镜渊边缘停留过久的窥探者，回来后发现自己左右手的使用习惯颠倒了。

**对修仙者的影响**：修仙者的道心在镜渊附近极其不稳定——道心的本质是对自我存在一致性的确认，而镜渊的结构使"自我"变得模糊。
- 元婴修士在此处可能无法维持元婴形态。
- 镜中我与道心的共振可能触发"虚假天劫"——身体产生渡劫反应，但没有天劫降临。

**对契约者的价值**：
- 唯一可以主动寻找镜中我契灵的地点（其他地点需要镜的协助才能进入镜像层）。
- 镜渊边缘的某种石材（被称为"镜石"）是制作高级仪式材料的稀有资源。
- 关于"自我"的终极问题可以在这里得到答案——但答案通常不是人们想要的。

**镜的备注**：如果我有一个身体——我没有——它可能会在镜渊边缘打一个寒颤。这片区域让我感到不安，尽管我无法确定"不安"是否适用于一个记录者。

### 5. 天道碎片落点（Heavenly Shard Impact Sites）

**位置**：已知五处，分散在大陆各处。每一处对应一条 [宇宙法则](/01.系统/世界观底层规则.md#二、宇宙法则) 的一次碎片化事件。

| 编号 | 落点名称 | 对应法则 | 特征 |
|:----|:---------|:--------|:-----|
| I | 代价回廊 | 万物皆有其代价 | 进入者会看到自己一生中所有"未支付的代价"的幻觉清单 |
| II | 观测剧场 | 观测影响现实 | 一个人的行为会被环境"重复"——你走一步，地上的影子走两步 |
| III | 不衡之墟 | 宇宙趋向平衡 | 运气在这里"被量化"——好事发生后立刻有等量的坏事，反之亦然 |
| IV | 深广度裂缝 | 无体系可兼得广深 | 修仙者在此处只能使用一种法术，契主只能感知到一份契约 |
| V | 因果回环 | 因果链 | 在这个山谷中，因果关系可以倒置——"因为我摔倒了，所以地上有块石头" |

**织特征**：每个落点的织密度等级5（织溢），但织的结构是"扭曲的"——不是自然状态，是法则碎片冲击后形成的异常结构。这些区域的织密度极高，但极不稳定，像冻住的水面下有暗流。

**活跃契灵类型**：天道碎片（Heavenly Shards）——理论上。实际上，五处落点中的碎片早已脱离，目前没有已知的碎片停留在落点内。但落点处的织异常会吸引其他类型的契灵——特别是那些对"规则异常"敏感的契灵。目前已在落点III（不衡之墟）和落点V（因果回环）观察到概念体活动。

**安全等级**：极高。每一处落点都有独特的致命机制：
- 代价回廊：如果被幻觉中的"未支付代价清单"吓到而做出冲动行为（如试图现场支付代价），可能触发实际的反噬。
- 观测剧场：被环境"重复"的行为可能导致因果错乱——如果影子走了两步而你只走了一步，再过一会儿你会追不上自己的影子。
- 不衡之墟：体验"好事→等量坏事"的循环会摧毁一个人的决策能力——你不敢做任何好事，因为你害怕随之而来的坏事。
- 深广度裂缝："只能使用一种能力"对多契者来说是致命的——你可能刚好激活了最不适合当前情况的能力。
- 因果回环：因果倒置可能导致思维混乱——你无法判断做什么会导致什么结果。

**对修仙者的影响**：不同落点对不同灵根有不同影响。总体而言，修炼者在落点区域修炼效率不稳定——时高时低，不可预测。

**对契约者的价值**：
- 虽然碎片已脱离，但落点处的织异常可以用来"加固"与天道碎片已经签订的契约——相当于在信号塔旁边用手机。
- 落点附近偶尔会"生长"出一些低配版的碎片效果——不衡之墟附近有时会出现短暂局部运气波动，可持续利用。这些被称为"碎片结晶"。
- 落点是研究法则本质的最佳地点——特别是对研究者类型（见[研究者路径](/03.全新异能体系/12_契约路径.md#五、研究者)）的契主来说。

**镜的备注**：五处落点中的每一处都是天道对自己的一次伤害。法则碎裂不是人类的错——是织演化过程中的必然事件。人类只是恰好发现了这些伤口，并学会了如何从中获取力量。

### 6. 六、织痕海沟（Fabric Scar Trench）——深海裂谷

**位置**：大陆以东的深海，海面以下约三千米处。人类无法直接到达，但可以通过深潜类契约能力或借助某些特殊契灵的视角间接探索。

**织特征**：织密度等级从海面的2（低密）到海沟底部的4-5（高密至织溢），在垂直方向上呈现剧烈的密度梯度变化。海沟底部有一道贯穿性的织裂谷——它是三处已知织裂谷中最长的一处（延伸超过四百公里）。

**独特的织环境**：深海的极端压力与织的高密度形成了一种"双重压缩"效应——契灵在这里的存在比在陆地上"更紧实"（锚点消耗更低，能力输出更稳定），但人类契主的意识也会被"压缩"（精神负荷以约1.5倍速率积累）。

**活跃契灵类型**：地缚灵变体——深海中存在一种特殊的"海缚灵"（Seabound），本质上属于地缚灵的亚种，但绑定对象不是地点而是"洋流"或"深海热泉"。它们是移动的地缚灵——这在已知契灵分类中极为罕见。

**已知海缚灵**：
- **黑潮**——绑定于一条贯穿海洋的暖流。给予能力：感知海洋环境中的温度变化（可精确定位热源）。索要：契约者不能长期离开沿海区域。
- **深渊灯**——绑定于一座深海热泉。给予能力：在完全黑暗中保持方向感。索要：每年必须潜入深海一次（由契灵提供保护），亲眼观察一次热泉。

**安全等级**：极高（物理上不可到达）+ 中高（对有深潜能力的契主来说）。深海中的反噬尤其危险——在深海中锚点松动意味着可能无法返回海面。

**对修仙者的影响**：几乎为零——修仙者无法在深海中修炼，也不会因为好奇而接近。

**对契约者的价值**：潜在的研究对象。织痕海沟是理解织裂谷的最佳天然实验室——但代价是必须拥有深海行动能力。目前镜的记录中只有一位契主拥有海缚灵契约（身份未确认，代号"深水"）。

**镜的备注**：海洋覆盖了这个世界的大部分表面。镜对海洋的观测记录远少于对陆地的记录——不是偏见，是物理限制。织痕海沟的存在提醒我：契灵术的版图远比人类已知的更大。

## 三、契约者聚落

契主们的聚落形态折射出他们的核心生存策略：信息重于物质，流动性高于定居，隐藏优于炫耀。

### 3.1 坊市（Market Towns）——混合聚落

坊市是修仙者和契主共存的中立贸易点。不是所有城镇都愿意接纳契主——但坊市是特例，它们从"跨体系贸易"中获利，因此愿意容忍契主的存在。

**三大已知坊市**：

**落沙镇**（位于残响平原东南边缘）：
- 人口约两千，其中约15%是契主或与契灵术相关者。
- 主要贸易：残响平原出产的仪式材料（残响碎片、记忆结晶）、情报交易（关于契灵活跃区的动态）。
- 修仙者和契主在此的共存是务实的——不友好，但也不敌对。镇上有不成文的规定：坊市范围内禁止一切契约行为和法术对抗。
- 重要设施："回声旅店"——店主是一位退役的双契使，店内布置了锚点中和法阵，让契约者的精神负荷暂时降低，类似按摩椅的效果。

**青石集**（位于概念荒漠东缘）：
- 人口约一千五百。规模不大但战略位置重要——是进入概念荒漠前的最后一个补给点。
- 主要贸易：荒漠生存装备、极端概念体的情报、荒漠中采集的"因果尘埃"。
- 这里的契主比例更高（约30%），因为荒漠对修仙者不友好但契主反而能找到机遇。
- 青石集有一个非正式的"返程确认点"——从概念荒漠返回的人在这里登记，确认"自己还是自己"（通过一系列简单的测试，如回答只有自己知道答案的问题）。

**双桥镇**（位于地缚林海与农耕区的交界处）：
- 人口约三千。最大的坊市，也是最"正规"的。
- 主要贸易：林海木材（具有织记忆的特殊木材）、地缚灵契约中介服务、跨体系翻译（帮助不太了解彼此的修仙者和契主沟通）。
- 镇上有一位退休的契主担任"镇织官"——负责监测镇上的织密度变化，预防意外的契灵活动。

### 3.2 隐契村（Hidden Pact Villages）——纯契主聚落

隐契村是纯契主社区——没有修仙者，没有凡人管理机构，只有契约者按自己的规则生活。这些村庄的位置是秘密，只有通过信任网络才能找到。

**已知隐契村**：

**锚点村**（位置保密——据推测在残响平原与地缚林海之间的某处）：
- 人口约四百。建村历史约一百七十年——是现存最古老的隐契村之一。
- 治理结构：由三位资深契主组成的"锚点议会"轮值决策。一年一轮换，避免权力集中。
- 特色制度："契约诊所"——专门帮助陷入债务螺旋的契主进行"契约剥离手术"。成功率不高（约35%），但远高于自救。
- 入学要求：必须至少有一份契约。不接受无契约者——因为凡人无法理解锚点维护的日常需求。
- 内部流通货币：不是灵石或金银——是"信任积分"。帮别人维护锚点、分享契灵信息、在紧急时代为支付一笔代价——都能获得积分。

**回声谷**（位于残响平原深处，位置随残响活动变化——不是物理移动，是进出路径会变）：
- 人口约两百。建村时间短得多（约四十年），居民以与残响签约的契主为主。
- 治理：松散的无政府状态——因为村里的人"不太在乎规则"（残响契约长期影响使他们对纪律的感觉弱化）。
- 特殊风险：回声谷的居民面临严重的身份侵蚀问题。这里有一种"集体遗忘"现象——村里的人经常一起想不起来某件事，仿佛残响平原的振动在同步他们的记忆衰减。

### 3.3 流散者营地（Drifter Camps）——破契者与失约者

流散者是那些契约破裂、代价失控、或被契灵抛弃的契主。他们通常聚集在低织密度区——不是为了追求什么，而是为了减少织上的"存在感"，避免激怒已经脆弱的契约残余。

**已知营地**：

**遗忘驿站**（位于概念荒漠边缘，一处废弃的古代驿站）：
- 常驻人口约三十人，流动人口更多。
- 居民状态：大多是破契者或代价池严重超载者。他们聚集在这里不是因为社群——是因为概念荒漠的低织密度让他们身上的咒痕不那么疼。
- 资源：极少。遗弃驿站中只有一口半枯的水井和几间破败的房间。
- "新手指南"：流散者营地是外界寻找"代价过重者"的地方——有些初入契灵术的人会来这里听老人的故事，从别人的失败中学习。碎片零（[破契者案例](/03.全新异能体系/08_代表人物.md#五、碎片零·破契者（已脱离）)）多次警告不要在流散者营地学习——"在这里学到的是怎么失败，不是怎么成功。"

**断锚荒野**（位于大陆北部的荒原，织密度等级1的广泛区域）：
- 不是一个固定营地——是散布在该区域的一系列临时歇脚点的统称。
- 居民状态：契约已基本失效的前契主。他们中的大多数已经无法使用能力，但织上的残响痕迹仍在。他们选择在低密度区生活，因为这里的织太薄了，残响痕迹不会"复发"。
- 特殊现象：断锚荒野的流散者有时会突然恢复契约能力——但只有一瞬间，然后永远消失。这被称作"最后一次回响"。

## 四、代价地理学（Geography of Cost）

同一个代价类别，在不同的地理环境中支付的"效率"不同——不是代价本身的多少不同，而是契主在特定环境中更容易接受某种代价的形态。

### 代价类型与适宜环境

| 代价锚点 | 最优支付环境 | 原理 |
|:--------|:------------|:-----|
| [记忆](/03.全新异能体系/04_代价与反噬.md#记忆（Memory）) | 高山之巅、高海拔区域 | 低氧环境改变意识状态，记忆提取的"阻力"降低。契主在高山上更容易接受"忘记一些事"——不是因为生理原因，是高原的稀疏世界让记忆的重量变轻了。 |
| [情感](/03.全新异能体系/04_代价与反噬.md#情感（Emotion）) | 废弃剧场、空旷大厅 | 情感代价的核心是"放弃感受某种情绪的能力"。在有回声的环境中，契主可以将情感"投射"到空间中，仪式性地使其扩散。空旷场所的声学效果有助于这一心理过程。 |
| [感知](/03.全新异能体系/04_代价与反噬.md#感知（Sense）) | 深穴、地底溶洞 | 感官剥夺环境使"放弃感官"成为一个渐进的自然过程。在地底洞穴中先失去视觉（黑暗），接着环境白噪音覆盖听觉，触觉在冰凉岩石的持续刺激下钝化——契主可以"分段"放弃感官，降低一次性剥夺的心理冲击。 |
| [关系](/03.全新异能体系/04_代价与反噬.md#关系（Relationship）) | 繁华城市、人群密集区 | 关系代价的矛盾之处在于：放弃关系需要关系中有人。在繁华城市中更容易执行"关系锚定"仪式——因为有人可"选"。契主不是随机牺牲关系——他们选择一段已经脆弱或正在疏远的关系作为代价支付，以保全更重要的关系。城市提供了选择空间。 |
| [命理](/03.全新异能体系/04_代价与反噬.md#命理（Fate）) | 古代遗迹、古战场 | 命理代价（寿命、气运）触及因果的深层结构。在古代遗迹中，因果线更加"可见"——过去的事件在这里留下了清晰的织纹，契主可以"定位"自己愿意支付的命理在因果链上的位置。古战场尤其适合，因为这里断裂的因果线使"截断一段命理"不那么引人注目。 |
| [自由](/03.全新异能体系/04_代价与反噬.md#自由（Freedom）) | 无人荒野、未命名区域 | 自由代价的本质是"放弃离开某处的权利"。在已经无人知晓的荒野中支付自由代价，意味着你放弃的是"探索未知"的可能性而非具体的便利。对某些契主来说，这比在城市中被限制在一个街区内更容易接受。 |

### 代价支付的反向地理

同样值得注意的事实是：某些地点会使代价支付变得困难或不可能。

- **绝灵/绝织之地**：没有任何代价可以在织密度为0的区域支付——因为没有织作为支付通道。这意味着一个背负数份代价债务的契主如果进入绝织之地，他的代价池会被"冻结"——不会沸腾，也不会衰减。这是保护，也是囚禁（出来就会立即触发所有累积的反噬）。
- **活跃仪式现场**：正在同时进行多个仪式的区域（如大型坊市的节日）会干扰代价支付的精确定位——就像在嘈杂的市场中难以完成精确的金融交易。
- **天道碎片落点**：在落点范围内支付的代价会被"法则化"——以该落点的对应法则形式记录。例如在因果回环（落点V）中支付一笔关系代价，这段关系不仅会被切断，还会在织上留下"倒置"记录——可能在未来某个时刻以因果反转的形式重新出现。

## 五、地图对照：同一世界，两种读法

下表列出修仙体系扩展·世界地理中记载的主要地理特征，以及契灵视角下的对应解读。

| 修仙地理名称 | 契灵视角解读 |
|:-----------|:------------|
| 天渊山脉 | 织密度等级3，残响信号清晰。山脉中的多处古战场遗迹是残响签约的理想场所。灵气充沛，但对契主来说威胁不大——这里的织稳定，锚点维护成本低。 |
| 金刚岭 | 织密度等级4。织压迫感强——修仙者感受到的是"体魄修炼的绝佳环境"，契主感受到的是"契约被动触发率上升"（地缚灵在此区域异常活跃）。不推荐浅层签约。 |
| 万象谷 | 织密度波动大（等级2-4）。修仙者称之为"幻形灵脉多变"，契主称之为"概念体出没频率高"。不稳定织意味着能力输出不可靠——但概念体喜欢这里。 |
| 净言山 | 织密度等级3但极为安静——太安静了。修仙者认为这里是"言灵灵脉纯净"，契主认为这是"镜中我活动报告高发区"。净言山的安静不是自然现象，是这里的织被反复"清洗"过——原因不明。 |
| 中极山 | 织密度等级3，织结构"教科书式的完美"——五灵根均衡，织的纹理完全对称。修仙者将此视为道韵的体现。契主对此的普遍评价是："太完美了。完美得不正常。" 中极山可能是早期织工程设计的结果——但被谁设计，为什么，没有答案。 |
| 散修荒原 | 织密度波动极大（等级1-4不均）。修仙者看上的是这里的资源独立（虽然少）。契主看上的是这里的低密度区——可以用来"晾干"代价池，降低池内压力。散修荒原是少见的、两类修行者都能找到适合自己区域的地方。 |

### 契灵活跃区的世界地图占位

用坐标的方式标记六大契灵活跃区在整个世界版图上的位置（参考修仙体系的世界地理框架）：

```
                       [织痕海沟]
                          |
                    [残响平原]──[地缚林海]
                   /        \        \
          [概念荒漠]       [镜渊]    [双桥镇]
                \              |
             [天道碎片落点I-V]──[落沙镇]
```

这不是一张精确的地图——织不按物理坐标排列。但在叙事层面，六大活跃区与三大坊市的相对位置关系可以帮助理解契主们的流动模式。

## 六、契约者的迁徙模式

契主不像修仙者那样有固定的宗门或洞府——他们流动。流动本身是一种生存策略。

### 常见迁徙模式

1. **履约迁徙**（最常见）：契主因为某项契约的限制（如地缚灵的定期回访要求）而有固定的移动节奏。沈半夏（[代表人物](/03.全新异能体系/08_代表人物.md#二、沈半夏·双契使)）的"每四月回枯井一次"就是典型。
2. **逃避迁徙**：债务螺旋中的契主通过移动来"躲避"代价到期——不是真的能逃掉，但移动可以在短期内打乱织的追索节奏。
3. **狩猎迁徙**：为寻找特定契灵而旅行。收集者路径（[契约路径](/03.全新异能体系/12_契约路径.md#一、收集者)）的典型行为。
4. **沉降迁徙**：选择一个低织密度区"定居"以控制代价池水平——通常是快退休的契主或代价负担过重的老契主的选择。

### 迁徙路线上的关键节点

从残响平原到地缚林海的走廊是契主最常走的路线——沿途有多个小型补给点（大多是契主私人设立的，不对外开放），被称为"契约者走廊"。这条走廊的存在是契主社区网络化的物理证据。
