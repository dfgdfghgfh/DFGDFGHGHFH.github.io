# 契灵术 · 代价与反噬

> 每个契主都会在某天凌晨突然坐起来，想起自己签过的某份契约的代价是什么。
> 那一刻的笑声——比哭更难听。

---

## 一、代价总论

代价（Cost/Price）是契灵术的第一推动力，也是最终的收盘者。没有代价就没有契灵术。忘记代价的存在，契灵术就会用更激烈的方式提醒你。

### 代价的本质

代价不是"法力消耗"或"精神力支出"。代价是**织上资产的转移**。你拥有某些东西在织上的所有权——记忆、情感、感知、关系、寿命、自由——契灵需要这些资产的一部分。你把它们转移给契灵，契灵把能力转移给你。

这是一笔真正的交易。不是"消耗"，是"转移"。

### 代价的支付方式

| 支付方式 | 说明 | 优点 | 缺点 |
|:--------|:-----|:-----|:-----|
| 一次性（预付） | 签约时全额支付 | 后续无负担，契约最稳定 | 风险集中，可能无法承受 |
| 分期（按周期） | 每次履约时支付一小部分 | 风险分散，可适应 | 总代价更高（有利息），需要持续履约 |
| 按使用计费 | 每使用一次能力支付一次 | 灵活，不用不付 | 单次使用成本高，容易在不注意时累积 |
| 触发式 | 特定条件触发时才支付 | 平时无负担 | 触发不可控，可能在最糟糕的时候扣款 |
| 混合 | 以上组合 | 灵活度高 | 管理复杂，容易漏算 |

## 二、六大代价锚点详解

### 记忆（Memory）

**付出什么**：你的记忆。不是"遗忘"——是"转移"。你失去的记忆会在契灵那边存在，但对你来说就是消失了。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 遗忘近期琐事：昨天吃什么、上周见过谁、钥匙放哪儿了 | 日常不便 |
| 轻度 | 遗忘重要信息：某人的脸、自己的生日、重要承诺 | 人际关系受损 |
| 中度 | 遗忘技能知识：忘记某种修炼方法、忘记一门语言 | 能力下降 |
| 重度 | 遗忘身份记忆：自己的名字、父母的脸、全部童年 | 身份危机 |
| 完全 | 遗忘自我：不知道"我是谁"，仅保留基本生存本能 | 自我毁灭 |

**可恢复性**：极难。织上的记忆转移是不可逆的——记忆已经被写入了契灵的存在结构中。理论上，如果你能找到接受你记忆的契灵并进行反向交易（用其他代价换回记忆），有可能部分恢复。但反向交易的代价通常比原始交易更高——因为契灵已经"消费"了你的记忆，要让它"吐出来"需要额外补偿。

**案例**：一个与残响签约三年的契主发现自己忘了妻子的名字。他记得她的脸、记得她们的故事、记得所有情感——但名字不见了。他妻子哭了整整一天。三天后他想起来了——因为他的一个梦里有残响原主的人生片段，其中提到了一个女性的名字。他不知道自己想起来的是妻子的名字还是残响记忆中某个人的名字。他到死都无法确认。

---

### 情感（Emotion）

**付出什么**：你的情感能力——不是某一个情感，而是情感的通路被部分切断或钝化。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 特定情感变弱：不容易被某一类型的事情触动 | 个性微调 |
| 轻度 | 一组相关情感钝化：对社交场合的情感反应下降 | 人际关系疏离 |
| 中度 | 一个完整的情感通道关闭：无法产生"嫉妒"或"兴奋" | 人格改变 |
| 重度 | 多个情感通道关闭：基本只剩下平静和偶尔的焦虑 | 情感残疾 |
| 完全 | 情感全面丧失：对一切无感，包括自己的处境 | 非人化 |

**可恢复性**：部分可逆。与记忆不同，情感能力更像肌肉萎缩而非切除——长期不用的情感通路可以重新激活，但需要时间和耐心。恢复方式包括：强烈的新情感体验（创伤或极乐）、能激活旧通路的记忆回溯、或通过其他契约借用情感能力（但这是债务螺旋的典型入口——见下文）。

**案例**：与「沉默」概念体签约的老默（见[案例](/03.全新异能体系/03_契约类型.md#沉默的守门人)）最早丢失的不是沉默，而是"对沉默的不耐烦"。那是情感中最微妙的一项——他不再在安静场合感到需要找话说的焦虑。三年后，他失去了"兴奋感"——成功完成一件事后的那种轻微快感。他不再庆祝。五年后，他的妻子发现他在母亲葬礼上表情平静。他不是不难过——他无法表达难过。

---

### 感知（Sense）

**付出什么**：五感之一被削弱、扭曲或剥夺。不是物理器官的损坏——是织上的感知通道被关闭或转接。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 某一感官敏锐度下降30%左右 | 需要适应，整体影响不大 |
| 轻度 | 某一感官持续低水平异常（如味觉中所有食物带金属味） | 生活质量下降 |
| 中度 | 某一感官阶段性失灵（每天有几小时看不见或听不见） | 严重影响日常生活 |
| 重度 | 某一感官被永久剥夺 | 残障 |
| 完全 | 多种感官被剥夺 | 彻底感官隔绝 |

**可恢复性**：理论上可逆——高等契灵术有"感官赎回"的记载，但成功率极低（约12%），且需要找到愿意做反向交易的高阶契灵。恢复后的感官通常在头几个月异常敏感，然后才回到正常水平。有些案例中恢复后的感官和原来不同——失去"视觉"的人重新获得视觉，但看到的世界是"另一种颜色"（波长偏移）。

**案例**：一个与"回声"概念体签约的契主以"味觉"作为代价。签约后他再也尝不出甜味——所有食物在他嘴里只有咸、酸、苦、鲜。他开始疯狂食用甜食但感觉不到任何满足。一年后，他的身体因为过量摄入糖分而出现健康问题——身体尝到了甜，意识没有。这给他带来了一个奇怪的处境：明知不应该吃甜食，但大脑不停寻找那个永远得不到的"甜味信号"，形成了一种生理性的成瘾行为。

### 关系（Relationship）

**付出什么**：一段特定的关系。这不是"忘记那个人"——是织上那条连接你和另一个人的因果线被削弱或切断。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 某段关系变得"有距离感"——不再能自然相处 | 关系降温 |
| 轻度 | 信任感下降——不自觉地怀疑对方 | 关系紧张 |
| 中度 | 关系中的核心连接断裂——不再理解对方的感受 | 实质性断裂 |
| 重度 | 关系被织上切断——对方不再把你视为有关联的人 | 社会性死亡 |
| 完全 | 该关系在因果层面被删除——仿佛从未存在过 | 存在抹除 |

**可恢复性**：不一定——取决于对方。关系是双方的，一方切断织上连线后，另一方可能仍然保持记忆和情感。但如果代价等级达到"重度"以上，对方会"不自觉地疏远你"，甚至不知道为什么。恢复需要双方的共同努力，且代价方（契主）需要通过长期的行为重建织上的因果线。

**案例**：一个女性契主为了获得一笔关键契约的有利条款，以"与母亲的关系"作为代价支付。代价等级是轻度——她以为自己能接受。

签约后三个月，她发现自己不再主动给母亲打电话。半年后，母亲打电话来时她感到不耐烦。一年后，母亲去世——她站在葬礼上，悲伤得撕心裂肺，但引起她悲伤的不是"失去母亲"而是"我不知道为什么我最后一年没有和她好好说话"。

关系代价常常这样运作：不是夺走你的爱——是夺走你爱的行动能力。

---

### 命理（Fate）

**付出什么**：你的寿命、运气、或因果链上的其他资产。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 微霉运：经常丢东西、迟到、小磕碰 | 日常增烦 |
| 轻度 | 可测量的寿命缩短（约1-3年）+ 偶发中等霉运 | 隐患 |
| 中度 | 寿命缩短5-10年 + 关键节点上的霉运 | 重大影响 |
| 重度 | 寿命缩短10年以上 + 持续的厄运期 | 可能致死 |
| 完全 | 因果线上出现"断点"——生命随时可能因为一个小意外终结 | 必死 |

**可恢复性**：一般不可逆。命理代价是永久性的——织上的因果链一旦发生偏移，很难恢复原状。寿命缩短就是缩短了，已经"发生"了。因果断点的修复尝试本身就是一件高危险的事——曾有契主试图用一片"因果断点"天道碎片修复另一份契约造成的命理代价，结果制造了一个因果悖论，导致他自己被从当天的事件线上"抹去"了六小时（他记得那六小时的一切，但任何记录——包括镜的记录——都显示那六小时他不存在）。

**案例**：一个名为"九爷"的老契主，年轻时为了获取一笔救命钱（他女儿重病需要一种极其昂贵的药），与一个他不确定类型的契灵签订了仓促契约。他没有仔细看代价条款——代价是"以等价寿命交换等价金钱"。

他女儿用那笔钱治好了病。九爷活到了73岁——在家族中算正常寿命。但他的牙齿在40岁开始脱落，50岁全口假牙；他的头发在35岁全部变白；他55岁时双目白内障。没有大病，身体的每一个器官都在正常老化——只是比同龄人快了一倍。

九爷说："那笔钱我从金额上还清了。但利息——我还在付。"

---

### 自由（Freedom）

**付出什么**：行为自由或地域自由——你的行动范围被限制，或某些选择被关闭。

**代价层次**：

| 等级 | 表现 | 影响 |
|:----|:-----|:-----|
| 轻微 | 离开某区域12小时以上出现轻度焦虑 | 出行受限 |
| 轻度 | 特定行为被禁止（不能吃某种食物、不能说某个字）| 生活不便 |
| 中度 | 被锁定在特定地理范围内（如一座城市） | 实质囚禁 |
| 重度 | 不能离开特定建筑/房间 | 严重囚禁 |
| 完全 | 锚点固定——只能在一个小范围内活动 | 终身监禁 |

**可恢复性**：可解除——通过解除契约。但解除契约本身可能有反噬。自由代价的独特之处在于：它可能随时间恶化。初始限制是"每月只能离开这个镇子三次"——如果长期不管理，可能演变为"无法离开这个镇子"。

**案例**：见[契约者丙](/03.全新异能体系/01_体系起源.md#镜的第一次直接观察)——一个完整的地缚灵自由代价案例，涵盖十年间的渐进恶化过程。

## 三、反噬（Backlash）

### 触发条件

反噬在以下任一条件下触发：

1. **契约破裂**：一方明确违约。最常见——契主未支付到期代价。
2. **锚点崩坏**：锚点因外部冲击（如其他契约意外干涉）或内部衰竭（太久未确认）而断裂。
3. **代价池超载**：未完全消耗的代价积累超过契主当前的承受阈值。代价池不是无限的——每个契主有一个隐性的容量上限，超过即反噬。
4. **实体背叛**：极少数情况下，契灵主动破坏契约——这不是违约而是"撕约"。这极其罕见，因为契灵同样要承受反噬——但某些契灵可能认为"承受反噬来换取自由"是值得的。

### 反噬严重度分级

| 等级 | 表现 | 恢复周期 | 致死率 |
|:----|:-----|:---------|:------:|
| **轻微** | 能力暂时失控（能力间歇性爆发或消失），24-72小时不适 | 3-7天 | 0% |
| **轻度** | 代价追加（一次性扣除相当于3-6期分期量的代价），伴随剧烈疼痛 | 1-4周 | <5% |
| **中度** | 代价加倍 + 咒痕出现（见下文）+ 至少一项能力永久丧失 | 1-6个月 | 15% |
| **重度** | 多重代价被同时追索 + 意识模糊 + 锚点连锁崩塌 | 6个月以上或不可逆 | 40% |
| **致命** | 织上锚点暴力断裂 + 契主存在本身被织"拉扯"——死亡或扭曲 | — | 80%+ |

### 咒痕（Curse Mark）

反噬在身体或灵魂上留下的可见/可感印记。咒痕是代价在织上累积后的"溢出"——当代价池容量不足以容纳更多的待支付代价，织会把一部分代价以物理形态"展示"出来。

常见咒痕类型：

| 类型 | 形态 | 含义 |
|:----|:-----|:-----|
| 织纹 | 皮肤上出现编织状的纹路，颜色从浅灰到深黑 | 代价池接近容量上限 |
| 空洞 | 身体某处出现"看起来没问题但感觉不存在"的区域 | 该处的感知已被部分转移 |
| 回声 | 说话时偶尔出现轻微回音（即使环境吸音） | 存在两个锚点正在干扰 |
| 裂口 | 皮肤上出现不流血的裂缝，透过裂缝可以看到"另一种颜色" | 织的结构在身体上显现 |
| 镜像偏位 | 镜中倒影与本人有微小差异 | 镜中我契约不稳定 |

咒痕被视为契灵的"从业证明"。在契灵术社区中，没有咒痕的契主被认为"不够认真"。但这是一种危险的浪漫化——咒痕出现意味着代价池已经进入了危险区。

### 反噬的传导机制

反噬不是随机的惩罚。它有明确的传导路径：

```
触发事件 → 锚点振荡 → 织的局部结构扰动 → 代价池沸腾 → 
契主在织上的位置偏移 → 物理/精神症状 → 连锁效应（影响其他契约）
```

理解这一传导路径可以帮助契主在早期介入：

1. **锚点振荡期**：第一个信号。契主会感到契约能力输出不稳定——忽强忽弱。此时介入可以进行"锚点加固"——再次确认契约，重置锚点位置。
2. **代价池沸腾期**：第二个信号。出现生理症状——头痛、恶心、睡眠紊乱。此时介入需要进行"代价池排空"——主动提前支付未来期数的代价，降低池内压力。
3. **位置偏移期**：第三个信号。契主开始感到"自己不在正确的位置"——一种本体感丧失。此时介入已经很困难，需要专业契主进行"归位仪式"。
4. **连锁效应期**：一旦进入，几乎无法阻止。此时的目标不是阻止反噬——是**止损**：牺牲受影响最严重的契约以保护其他契约。

## 四、债务螺旋（Debt Spiral）

### 定义

债务螺旋是契灵术中最致命的陷阱之一：**用一份契约的能力或代价去支付另一份契约的代价**。

### 典型路径

1. 契主 A 与契灵 X 签订了契约，代价是每月支付一段记忆。
2. 某个月 A 没有足够的记忆可付（或者不愿意），于是他启动了他的另一份契约 B 的能力——"代价转换"——将记忆代价转为情感代价。
3. 契约 B 的能力被使用了，需要按照使用频率支付代价。使用 B 的代价是……一段记忆。
4. A 发现自己用 B 来支付的正是 B 本身索取的代价——他在用左口袋的记忆填右口袋的记忆漏洞，而每次转移都有损耗。
5. 三个月后，A 的两个契约的代价池都沸腾了，他试图用第三份契约 C 来平账——然后三份一起崩。

### 螺旋特征

- 进入螺旋的人几乎不会主动退出（因为退出意味着立刻承认损失）。
- 螺旋的加速是非线性的——初期几个月看不出问题，然后突然崩盘。
- 螺旋中的契主会对人隐瞒自己的真实状态，因为承认螺旋意味着前面所有的代价支付都变成了"错误投资"——这涉及沉没成本谬误。
- 螺旋唯一已知的退出方式：**主动违约至少一份契约，承受一次重度反噬，然后用残留的清醒处理剩下的事**。相当于断臂求生。

### 镜的观察

> 债务螺旋不是技术问题。是心理问题。契主在螺旋中做的每一个决定单独看都很合理——"先用B救急，下次再还"。问题是"下次"永远不会来。
> 这和人类的信用卡债务模式完全一致。织只是复制了你们自己发明的陷阱。
