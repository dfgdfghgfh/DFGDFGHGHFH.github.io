# 契灵术 · 能力树

> 修仙者问："你的境界是什么？"
> 契主回答："我有三份契约。"
> 修仙者不理解。这不是他的体系框架——增长不是提升，是扩展。

---

## 一、核心理念：水平扩展而非垂直升级

契灵术的成长不是梯子——是书架。

修仙的境界体系是一把梯子：你从凡人层爬到炼气层再爬到筑基层——越来越高，但能站的空间越来越窄。

契灵的成长是往书架上添加更多的书（契约）：你不一定爬得更高，但你读得更多。每加一份新契约，你获得一个新的能力维度。书架不会让你更接近天花板，但它让你能处理更多类型的任务。

**这不是说契灵术没有深度。深度在于契约的组合和协同——不在于单一能力的上限。**

---

## 二、能力类别

契灵术的能力分为五大类。每类对应一种操作类型，而非与契约类型一一对应。一份契约可能同时提供多个类别的能力（但深度不及专精类）。

### 1. 感知类（Perception）

对织、能量、存在的感知能力。不是物理感觉的延伸——是对织上信息的读取。

| 能力 | 来源典型契约 | 效果 |
|:----|:------------|:------|
| 织视 | 残响/地缚灵 | 看到织上的锚点和波动（类似热成像但显示的是织密度） |
| 存在感知 | 地缚灵 | 感知附近是否有其他契约存在（但不识别具体内容） |
| 概念嗅觉 | 概念体 | 感知周围环境中"哪个概念的浓度最高" |
| 真知揭示 | 镜中我 | 一个瞬间看清事物的真实面目（一天限用一次） |
| 法则感知 | 天道碎片 | 感知附近是否存在法则层面的异常 |

### 2. 干涉类（Intervention）

对物质界或织进行直接操作的能力。最消耗代价的能力类别。

| 能力 | 来源典型契约 | 效果 |
|:----|:------------|:------|
| 领域微调 | 地缚灵 | 在绑定地附近改变环境参数（温湿光声） |
| 概念干预 | 概念体 | 对某个概念进行小幅度操作（让一道边界暂时软化） |
| 法则微调 | 天道碎片 | 在极小范围内暂时调整法则的运作方式 |
| 因果介入 | 天道碎片 | 在因果链上施加一个微小推动（危险操作） |
| 形态借用 | 地缚灵（深契） | 临时借用绑定地的物质形态 |

### 3. 召唤类（Summoning）

不完全召唤契灵本身——而是借调契灵的一部分存在到物质界。

| 能力 | 来源典型契约 | 效果 |
|:----|:------------|:------|
| 残响投影 | 残响 | 将残响的部分意志投影到物质界——可获得战斗协助 |
| 地缚具象 | 地缚灵 | 将地缚灵的意识在绑定地短暂具象化（如让枯井翁在井口现身） |
| 概念化身 | 概念体（极深契） | 让概念体以近似人类理解的形式短暂显现——极消耗代价 |
| 镜像召唤 | 镜中我 | 让镜中我在物质界出现（非常短暂，约10秒） |

### 4. 转化类（Transformation）

对自己或目标进行状态改变。契灵术中最"像法术"的类别。

| 能力 | 来源典型契约 | 效果 |
|:----|:------------|:------|
| 技艺传承 | 残响 | 暂时获得一项不属于自己的技能（来自残响原主） |
| 状态覆盖 | 概念体 | 用概念的逻辑覆盖局部现实——如让一个区域"沉默" |
| 角色扮演 | 地缚灵（废墟剧场） | 临时借用他人的织属性——成为另一个人 |
| 自我整合 | 镜中我（深契） | 短暂借调镜像层的"纯净自我"——全状态临时提升 |

### 5. 认知类（Cognition）

对信息和意识的操作。最微妙也是最容易被低估的能力类别。

| 能力 | 来源典型契约 | 效果 |
|:----|:------------|:------|
| 记忆检索 | 残响 | 从记忆深处精确提取被遗忘的信息 |
| 连接感知 | 地缚灵（长桥） | 感知事物间的最近路径（不限于物理） |
| 概念视野 | 概念体 | 切换到以某概念为核心的感知模式 |
| 动机解析 | 镜中我 | 分析自己或他人行为背后的真实动机 |
| 信息检索 | 残响（守藏老人） | 在信息库中瞬间定位目标信息 |

## 三、成长路径：从单契约到多契约

### 阶段一：执契者（1份契约）

**定位**：入门。已经完成了一次签约——你不再是凡人了。

- 只拥有一个能力类别（取决于你的第一份契约类型）。
- 精神负担最小——几乎不影响日常生活。
- 锚点管理技能为零——你不知道如何维护契约，全靠新手运气。
- 典型问题：对代价的感知不准确。第一份契约的代价往往要到半年后才开始"感到重量"。

**建议路径**：资深契主通常建议第一份契约选残响或地缚灵——它们的代价可预测，反噬相对温和。概念体和天道碎片不适合作为初契。

### 阶段二：双契使（2份契约）

**定位**：可独立运作。开始理解契约管理的逻辑。

- 拥有2个能力类别——可能性开始出现。
- 精神负担轻微——需要开始注意休息和锚点维护节奏。
- 开始接触锚点规划——两份契约的锚点需要合理安排间距，避免互相干涉。
- 第一次面对契约协同的风险和收益。
- 典型问题：两份契约的代价支付时间表可能冲突——需要学会错峰支付和提前储备。

**协同策略**：

| 组合 | 协同效果 | 风险 |
|:----|:---------|:-----|
| 残响 + 地缚灵 | 地缚灵提供稳定的感知环境，残响提供战斗能力——攻守兼备 | 自由代价 + 记忆代价双线累加 |
| 概念体 + 镜中我 | 概念视野 + 自我认知——极高的信息处理能力 | 精神负担双倍，角色混淆风险 |
| 地缚灵 + 地缚灵 | 双地点覆盖——可以有两个"主场" | 锚点间距管理困难，自由代价累加 |

### 阶段三：多契者（3份契约）

**定位**：资深从业者。这是"舒适区"的上界。

- 拥有3个能力类别——可以形成三角支撑结构。
- 精神负担中等——需要刻意管理注意力，容易疲劳。
- 锚点布局进入专业领域：三个锚点需要形成"不互相干扰但又能协同"的三角形结构。
- 第一个债务螺旋风险开始出现——三份契约的代价交织在一起，如果管理不善，一份契约的问题可能拖垮其他两份。
- 典型问题：代价池容量开始接近上限——咒痕可能出现。

**三角结构示例**：

```
         残响（战斗）
        /           \
地缚灵（感知）——概念体（认知）
```

这个结构的优点是三类能力互不重叠，形成面覆盖。缺点是没有冗余——任何一份契约出现问题，能力体系就会出现缺口。

### 阶段四：契主（4份契约）

**定位**：安全上限。达到这个级别的人都是认真的从业者。

- 拥有4个能力类别——理论上覆盖所有基础操作类型。
- 精神负担高——长期处于轻度认知过载状态。
- 锚点管理成为全职工作——需要定期检查和微调每个锚点的状态。
- 需要完善的代价管理策略——代价池随时可能沸腾。
- 需要至少一条"逃生路径"——一份可以快速安全解除的契约，以便在其他契约出问题时释放容量。

**四位一体结构（推荐）**：

```
        感知类（地缚灵/残响）
        /        |        \
   干涉类(天道)  认知类(镜中我)  转化类(概念体)
```

四份契约各有明确分工，互不重叠。最稳定的结构——但也是最难建立的，因为需要找到四类互不冲突的契灵。

### 阶段五：破契者（5+份契约）

**定位**：这不是升阶——是堕入深渊。5+契约的契主没有一个能在长期维持稳定。

- 精神负荷远超安全阈值——需要药物或外部干涉才能入睡。
- 锚点冲突不可避免——5个锚点无论如何布局都会相互干扰。
- 至少1-2份契约处于"破损状态"——即不完全履约、不定期确认的状态。
- 代价池长期处于沸腾边缘——咒痕遍布全身。
- 平均寿命预期：签约5+后的存活率在15年内不足30%。

**为什么有人会进入这个状态**：大多数破契者不是"选择"了这条路——他们是先慢慢累加到4份契约，然后在某一次危机中被迫签下第5份"救命契约"；然后第5份契约的代价让他们不得不再签第6份来支付前一份的债务。这是债务螺旋的终局版本。

## 四、协同效应

当特定的契约类型同时存在时，可能产生协同效应——不是简单的叠加，而是质变。

### 已知协同公式

| 组合 | 效应名 | 效果 |
|:----|:-------|:-----|
| 残响 + 残响 | 双重回响 | 两份残响的技艺传承可以相互补全——能力完整度从单份的70%提升至约85% |
| 地缚灵 + 概念体 | 领地概念化 | 在绑定地范围内，概念体的能力获得稳定性加成（概念干预的效果提升约40%） |
| 镜中我 + 残响 | 记忆审计 | 镜中我可以帮助区分"自己的记忆"和"残响注入的记忆"——降低身份侵蚀风险 |
| 天道碎片 + 天道碎片 | 法则共振 | 两份碎片在特定条件下可能产生法则叠加效应——但同时也使天道注意度翻倍 |
| 概念体 + 概念体 | 概念融合 | 两个概念体之间如果概念相邻（如"沉默"和"回声"），可能产生意想不到的新能力——完全不可预测 |

## 五、过度专精的风险

与修仙体系不同，契灵术不鼓励专精。过度专精（所有契约同一类型）在契灵术中反而是陷阱。

**全残响配置"收藏家"**：
- 优势：多项传承能力，战斗面广
- 劣势：记忆代价重叠，身份侵蚀加速——多份残响的代价可能产生"叠加效应"，遗忘速度不是线性是乘数
- 死亡风险：早期老年痴呆（实际年龄可能只有40岁）

**全地缚灵配置"地头蛇"**：
- 优势：在多个地点拥有主场优势
- 劣势：多份自由代价叠加——你可能同时被五个地点绑定，产生"没有一份自由属于自己"的状态
- 死亡风险：被困死——物理和精神上的

**全概念体配置"哲学家"**：
- 优势：多种概念视野，对世界的理解远超常人
- 劣势：人性稀释——每个概念体都在索取情感/感知代价，叠加效应让你迅速非人化
- 死亡风险："成为概念"——不再是人类

**全天道碎片配置"赌徒"**：
- 优势：法则级能力
- 劣势：天道注意度随碎片数量乘数增长——不是五倍的注意，是2^5=32倍的注意
- 死亡风险：被天道"修正"

## 六、推荐构建示例

### "收集者"（全残响流）

- **契约**：断剑陈（剑意）、药篓（丹方）、铁衣（防御）、守藏老人（检索）
- **能力覆盖**：战斗 + 炼药 + 防御 + 信息处理
- **代价负担**：记忆（轻度到中度）——身份侵蚀是持续威胁
- **战略**：依靠残响的丰富传承成为一个多功能战力。需要频繁使用记忆复原仪式延缓遗忘。
- **弱点**：面对没有"对应原主记忆"的新情况时缺乏应变能力

### "锚定者"（双地缚灵 + 残响 + 概念体）

- **契约**：长桥（路径感知）、晚钟（时间节律）、断剑陈（战斗）、边界（感知调控）
- **能力覆盖**：感知 + 时间管理 + 战斗 + 边界操作
- **代价负担**：自由（不能断关系）+ 行为限制（规律作息）+ 记忆（轻度）+ 边界模糊
- **战略**：利用长桥+晚钟的协同打造一个"永远在最正确的时间出现在最正确的地点"的定位——适合探索者、佣兵
- **弱点**：一旦离开已建立的地点网络，能力大幅下降

### "真知者"（镜中我 + 概念体 + 天道碎片 + 残响）

- **契约**：镜中我（尤女士款——真知揭示）、沉默（静心）、观测裂隙（盲点）、守藏老人（信息检索）
- **能力覆盖**：认知 + 洞察 + 隐秘 + 知识
- **代价负担**：心理压力（面对真相）+ 情感钝化 + 天道注意 + 记忆
- **战略**：成为一个信息型契主——收集情报、分析局势、做出最优决策
- **弱点**：几乎没有正面战斗能力。高度依赖环境和时机。
