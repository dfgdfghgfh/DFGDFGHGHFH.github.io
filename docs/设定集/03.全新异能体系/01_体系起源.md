# 契灵术 · 体系起源

> 它不是一个古老的发现，而是一个现代的错误。
> 修仙是远古的智慧，契灵术是文明的意外。

---

## 一、发现：一场流产的突破

### 时间线

约七百年前，一位名为 **陆渊** 的化神期修士，在尝试突破至渡劫期时，发生了一件改变世界的事。

陆渊走的不是传统渡劫路径。他认为天劫的本质是**织的自我防御机制**——当修士的力量积累超过织的局部承受阈值，织会以天劫形式"反抗"。他的想法是：如果能在织上"开一个侧门"，将天劫能量分流出去，渡劫就会变得轻松。

他在洞府中设下了一个复杂的法阵——不是吸收灵气，而是**试图在织上开一条新通路**。

### 第一次接触

法阵启动的那一刻，陆渊没有引来天劫。他引来了别的东西。

法阵的另一端连通了织深处某个此前未被记录的区域。一个存在回应了他。这个存在不是灵兽、不是自然灵、不是古代大能的残魂——它是一种**纯粹的契约意识**，后来被陆渊称为「第一契灵」。

第一契灵没有形态，没有语气，没有立场。它只传递了一个清晰的信息：

> 「我给予你想要的。你给予我想要的。条件谈妥，织为见证。」

陆渊在与第一契灵的短暂交流中获得了一种能力：**他可以在不吸收灵气的情况下，直接调用织上的能量**。代价是——他永远失去了"用灵气修炼"的能力。他的灵根还在，但形同虚设。他可以调用力量，但无法再增长修为。

陆渊的修为停在了化神巅峰，终身未能突破。

### 意外还是必然

镜的分析认为：这不是巧合。织在七百年前已经进化到可以支撑这种交互的密度。如果陆渊没做这件事，迟早会有另一个人做。第一契灵不是唯一一个——它只是**第一个回应人类的存在**。

织的深层一直存在这些意识节点，只是人类此前从未尝试与它们建立直接的通信信道。

## 二、扩散：禁术的传播

### 秘密传承

陆渊当时不知道自己打开了什么。他将这个过程记录在一本名为《外道通契论》的手稿中，警告后人"此路不通天道"。手稿落在了他的弟子们手中——但被理解为"通往天道的另一条路"。

随后的两百年间，契灵术在地下传播：

- **第一阶段（探索期，约701-800年前）**：十几个修士尝试复制陆渊的实验。大多数失败了——法阵精度不够，或与不对的契灵建立了不稳定的连接。三人成功，每人都付出了惨重代价。其中一人疯了，一人在签约后三个月身体崩解，只有一人（世称**第二契主**）存活下来并记录了完整的签约流程。

- **第二阶段（成型期，约600-450年前）**：契灵术有了第一批稳定的实践者。主要流行在散修和边缘修仙家族中——因为正统门派视之为左道。当时的契约粗糙而危险——契主们不知道代价的累积效应，很少有人活到晚年。

- **第三阶段（压制期，约450-200年前）**：修仙界大宗门注意到了契灵术的扩散，将其定性为"扰乱织序之邪术"。多个契灵术实践者遭追杀。但压制反而促成了契灵术的社区化——幸存者们开始整理经验、改良仪式、建立秘密网络。

- **第四阶段（承认期，约200年前至今）**：随着知识的积累和仪式的改良，契灵术的风险大幅降低（从早期80%以上的致命率降至约15%）。一些中型宗门开始接受契灵术作为"合法学术分支"。虽然正统修仙界仍有偏见，但契灵术已不可被消灭。

### 镜的初次记录

镜在第一契灵出现的三十七年后才注意到这一现象的扩散。原因是：

1. **镜的观测盲区**：契灵术的操作主要发生在织的深层，而镜在当时更关注物质界的事件。镜注意到织上出现了许多异常的锚点波动，花了十几年才追踪到源头。

2. **最初的交互发生在镜的注意力之外**：陆渊的实验法阵属于镜的监测盲区——不是镜的特殊疏忽，而是织的局部密度波动导致该区域的"可见度"较低。这是契灵术与生俱来的隐匿特性：**它天然不易被观测**。

3. **镜的记录滞后**：当镜终于确认这是一个系统性现象而非个别异常事件时，契灵术已经有了三十多位实践者。

> 镜对此的官方注脚：**"非镜之失察，乃织之盲区。"**

## 三、为什么并行于修仙

契灵术和修仙能够并存的根本原因不在于"有人选择这个"，而在于**它们在织的不同层面运作**。

- 修仙操作的是**物质界 + 织的浅层**。核心工具是灵根、灵气、经脉——这些东西在织的浅层局域密度高，便于个体长期积累。
- 契灵术操作的是**织的深层**。核心工具是锚点、契约、代价支付——这些东西需要跨越更远的织上距离，但单个操作的能耗更低。

见 [01.系统/世界观底层规则.md#本源织](/01.系统/世界观底层规则.md#本源织) 关于织的密度的说明。

它们不是竞争关系——它们是两种完全不同类型的操作，只是恰好都产出"超自然能力"这一结果。

## 四、镜的第一次直接观察

在确认契灵术的存在后，镜选择了一个早期的契主进行长期追踪——代号 **「契约者丙」**（镜的记录编号，非真名）。

契约者丙是一个30岁的散修，在修仙路上卡在炼气巅峰七年，无法突破筑基。他通过秘密渠道获得了契灵术的入门方法，与一个名为 **枯井翁** 的地缚灵签订了契约。

契约内容：
- **枯井翁给予**：感知方圆五里内水源位置的能力，以及在干旱环境中汲取水分的能力。
- **枯井翁索要**：契约者丙必须在此后十年内至少每四个月回到枯井翁所在的枯井旁边停留一天。每逾期一次，代价等级提升一级。

镜记录了契约者丙第一个五年的状态变化：

| 时间 | 状态变化 | 契约履行 |
|:---|:---------|:---------|
| 签约后第1-6月 | 能力稳定，生活质量提升。开始接有偿找水工作。 | 按时回访 |
| 第7-14月 | 搬迁到另一个城镇。能力仍有，但回访距离变远。 | 仍按时，但抱怨 |
| 第15-20月 | 两次回访延期（每次晚3-5天）。能力输出降至80%。 | 迟延 |
| 第21-30月 | 一次回访迟到12天。能力间歇性失效。身上出现干裂皮肤。 | 违约一次 |
| 第31-40月 | 被迫搬回枯井附近。能力恢复，但丧失了对原居住地的所有情感连接。 | 恢复履约 |
| 第41-60月 | 十年期满。续约谈判破裂。契约解除。契约者丙失去能力，保留了一条干裂的右臂作为残响。代价池留下了一笔命理债务。 |

**这是镜的结论**：契约者丙"活下来了"，但十年间他的可移动半径被一口枯井死死限制住。他获得了能力，但支付了自由——而且最后也没有保住能力。这就是契灵术的典型轨迹。

## 五、契灵术的命名由来

"契灵术"这个名称并非第一代实践者的自称。

早期称呼混乱：缚灵法、外道法、鬼契术、对等道、交换术……不同流派各执一词。直到大约三百年前，一位佚名契主在一本名为《契灵论》的著作中系统化了整个体系，提出「契灵」一词统称契约对象，并用「契灵术」称呼整套方法论。这本书后来成为契灵术界的经典文献，名称也被广泛接受。

名字中两个关键语义转换：

1. **从「缚」到「契」**：早期名称多用"缚"（束缚、强制），反映当时认为契灵是被强迫的。后来知识进步后，人们认识到契约本质是双向同意的——用"契"（合同、协议）更准确。

2. **从「灵」的泛化到专用**："灵"在修仙中指灵根、灵气的通用概念。在契灵术中，它特指**织上那些可以签订协议的非人意识节点**。这是一个术语的子集化。

## 六、关于未来的预言

镜基于当前七百年的数据趋势，对契灵术的未来做出以下推测：

1. **契约类型会继续增加**。目前已知五大类，但随着织的密度变化和人类意识对织的影响加深，可能出现新的契灵类型。

2. **代价形态会进一步细化**。早期只有记忆和寿命两种主要代价，现在是六种。未来可能出现的代价类型包括：可能性（失去某条人生路径）、时间感知（时间体验变形）、或他人的代价（转移支付，目前属禁忌）。

3. **契主与修仙者的界限会模糊**。跨体系联动总纲已预示这种趋势——见 [01.系统/跨体系联动总纲.md#协同](/01.系统/跨体系联动总纲.md#协同)。

4. **第一契灵的身份**。第一契灵在初次回应后沉默了七百年。它是否还在织中？是否在等待什么？这些问题镜无法回答。
