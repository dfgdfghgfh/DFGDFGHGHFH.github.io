# 契灵术 · 概述

> 不是向内挖掘，而是向外联结。不是积蓄，而是交换。不是苦修，而是博弈。
> 当修士在洞府中打坐吐纳时，契主在谈判桌前与神灵讨价还价。

---

## 一、名称哲学

**契灵术**（Spirit Pact Art）——三个字各有分量：

| 字 | 含义 | 哲学注脚 |
|:--|:-----|:---------|
| **契** | 契约、协议、凭证 | 力量来源于合约，而非修炼。这不只是技术分类，更是存在方式的宣言。 |
| **灵** | 灵性存在、织上的他者 | 力量不是孤立的。每一次契约都牵扯另一个存在——契约是关系，不是属性。 |
| **术** | 技艺、方法、交易策略 | 这不是"道"（修行之路），是"术"（实操技能）。契灵术不追求终极真理，追求有效的交换。 |

别名：

- **契约术** — 民间最常用叫法，偏重"签合同"的实操层面
- **缚灵之法** — 古代称呼，含贬义（缚 = 束缚，暗示此道不光彩）
- **对等之道** — 少数契主对自己的称呼，强调"等价交换"的核心理念
- **交易之道** — 修仙界对它的蔑称，暗指此路是"买卖人"的雕虫小技

镜的记录中使用「契灵术」作为标准称谓。

## 二、一句话定义

> **契灵术是通过与织上的非人存在签订具有法律级约束力的契约，以主动承诺特定代价为条件，换取即时可用的超自然能力的体系。**

这一定义的四个关键成分：

1. **契约** — 力量来源是合约，不是积蓄。没有契约就没有力量。
2. **代价** — 力量不是免费的。承诺在先，支付在后。违约即反噬。
3. **即时** — 签约瞬间获得能力，不需要长时间的修炼周期。这是契灵术对修仙最大的优势。
4. **织上非人存在** — 交易对象是织上的意识节点，不是随便什么灵体。

## 三、核心对比：契灵术 vs 修仙

> 完整对比见 [03.全新异能体系/10_与修仙的区别.md](/03.全新异能体系/10_与修仙的区别.md)，此处仅列骨架。

| 维度 | 修仙 | 契灵术 |
|:----|:-----|:-------|
| **核心行为** | 每日修炼、积累、突破 | 谈判、签约、履约、换约 |
| **力量来源** | 自身灵根 + 天地灵气 | 契灵 + 契约条款 |
| **成长曲线** | 缓坡 → 陡坡（前慢后更难） | 陡升 → 缓坡（快但浅） |
| **风险结构** | 前置（渡劫失败、心魔） | 后置（代价分期、反噬延迟） |
| **上限决定** | 天赋 + 自律时间 | 谈判能力 + 承受代价的意愿 |
| **退出成本** | 散功（痛苦但活着） | 反噬（可能直接死亡或扭曲） |
| **社会地位** | 正面、受尊敬 | 复杂（敬畏 + 戒备） |
| **核心隐喻** | 农耕——播种、浇灌、收获 | 贸易——谈判、签约、交割 |

## 四、体系概览

### 核心资源：契约（Pact）

契约是契灵术的一切。每个契主拥有的契约数量决定其等级。没有"经验值"或"修炼进度条"。只有合约的签订、维护、深化——以及可能的破裂。

### 契主等级（水平扩展，非垂直境界）

| 称号 | 契约数 | 说明 |
|:----|:------|:-----|
| 执契者（Pact Bearer） | 1 | 入门。第一份契约最危险——你还不懂代价的重量。 |
| 双契使（Dual Pact Envoy） | 2 | 可承受。两份契约可形成初步协同。 |
| 多契者（Multi-Pact Bearer） | 3 | 稳定与风险平衡点。三份是大多数契主的舒适区。 |
| 契主（Pact Master） | 4 | 安全上限。四份需要高度纪律和强大的锚点管理能力。 |
| 破契者（Pact Breaker） | 5+ | 超过安全上限。部分契约处于破损或不稳定状态。非升阶——是堕入深渊。 |

> 没有"境界突破"——契主不提升自身本质。他们只是不断扩展合约组合。
> 更多细节见 [03.全新异能体系/06_能力树.md](/03.全新异能体系/06_能力树.md)。

### 五大契灵类型

| 类型 | 本质 | 给予力量 | 索取代价 |
|:----|:-----|:---------|:---------|
| **残响**（Echoes） | 逝去修士的意志碎片 | 传承法术、战斗经验 | 记忆（身份侵蚀） |
| **地缚灵**（Bound Spirits） | 地点凝结的灵性意识 | 环境操控、领域感知 | 自由（地域限制） |
| **概念体**（Concepts） | 抽象概念的半觉醒灵体 | 概念干预、规则弯曲 | 安定（精神稳定性） |
| **镜中我**（Mirror Self） | 织上的平行自我投影 | 自我认知、真知之眼 | 面对不愿面对的真相 |
| **天道碎片**（Heavenly Shards） | 宇宙法则的碎裂片段 | 现实扭曲、因果干预 | 天道注意（不可预测性） |

详细展开见 [03.全新异能体系/03_契约类型.md](/03.全新异能体系/03_契约类型.md)。

### 六大代价锚点

| 锚点类型 | 支付内容 | 可恢复性 |
|:---------|:---------|:---------|
| 记忆 | 失去特定记忆 | 极难（织写入不可逆） |
| 情感 | 情感能力钝化或丧失 | 部分（需要时间或治疗） |
| 感知 | 五感之一被削弱或剥夺 | 理论上可逆（高等契灵术） |
| 关系 | 特定关系被削弱或断裂 | 不一定（取决于对方） |
| 命理 | 寿命缩短、运气变差 | 一般不可逆 |
| 自由 | 行为或地域被限制 | 可解除（解除契约） |

详细展开见 [03.全新异能体系/04_代价与反噬.md](/03.全新异能体系/04_代价与反噬.md)。

## 五、体系定位

契灵术在本宇宙中充当以下角色：

1. **快车道**：不想用十年修仙的人可以在此找到力量——但代价会在后面追上来。
2. **多样性供给**：修仙提供深度的专精能力，契灵术提供宽度的组合能力。
3. **风险对冲**：二者可在一定程度上互为"保险"——见 [01.系统/跨体系联动总纲.md](/01.系统/跨体系联动总纲.md)。
4. **叙事张力**：契约总有代价，代价总有显现的一天。每份契约是一个潜在的转折点。
