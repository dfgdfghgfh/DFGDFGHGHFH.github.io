# 跨体系联动 · 转换与借用

> 修仙者的灵力和契主的回响——一个是水，一个是冰上的纹路。
> 你可以把水结成冰纹，也可以让冰纹融化成水——但每一次形态转换，都有东西在过程中流失。
> 织从不允许 1:1 的兑换。

---

## 一、核心原则

[01.系统/跨体系联动总纲.md#第三定律转化总有损耗](/01.系统/跨体系联动总纲.md#第三定律转化总有损耗) 明确规定：

> 跨体系资源转化时，输出量 < 输入量。差额被织吸收。

本文件在这一原则下展开具体的转换路径、兑换率和风险。

## 二、转换通道

### 通道一：修为 → 契约深度

**方向**：修仙资源 → 契灵资源

**操作**：将自身修炼积累的修为属性（灵根经验、境界感悟）转化为契约锚点的深度。类似于"用修炼成果提前支付契约的未来代价"。

**机制**：
- 每次转化消耗一定量的修为经验（具体数值取决于当前境界和契约状态）
- 转化后，契约深度增加，契约能力输出增强
- 但该部分修为经验从灵根中永久扣除——境界不会倒退，但该境界的"厚度"（积累的冗余经验）减少

**兑换率**：约 60%-70% [待测试]
- 投入 100 单位的修为经验 → 契约锚点获得约 60-70 单位的深度增长
- 损耗的 30-40% 被织吸收

**限制**：
- 只能在突破后 30 天内进行——突破后的身体处于织重新编织期，较容易接受结构变化
- 每年最多转化 1 次，否则灵根结构会因频繁的修为提取而不稳定
- 转化不可逆——修为经验转化为契约深度后，无法再转回

**叙事含义**：修士将辛辛苦苦修炼的成果"烧掉"一部分来加固契约。每次转化后，修炼者会感觉到一种"空荡感"——不是境界下降，是积累的充实感减少了。

### 通道二：道心 → 锚点强度

**方向**：修仙资源 → 契灵资源

**操作**：将道心积累（存在一致性）转移到契约锚点上，增加锚点的织上稳定性。

**机制**：
- 道心不像修为那样有具体的"数值"——它更像是一种"结构质量"。转化时，道心的"一部分坚固性"被转移到锚点中
- 转化后，锚点的抗干扰能力上升（抵抗反噬和外部干扰的能力增强）
- 但道心自身"变薄"了——不是变弱，是在相同强度下变得更容易被穿透

**兑换率**：约 50%-55% [待测试]
- 转化后锚点稳定性提升量约为道心消耗量的 50%
- 损耗的 45-50% 被织吸收

**限制**：
- 道心转化有累积风险——每次转化都使道心结构多一条"接缝"。这些接缝在受到心理冲击时可能成为裂纹的起点
- 总转化次数上限：5 次（一生）。第六次转化会使道心结构进入不可逆的脆弱状态
- 转化方向不可逆——锚点中的道心成分不能转回道心

**叙事含义**：将自己的"存在一致性"割让一部分给契约。转化后，修炼者可能会发现自己在某些事情上变得不那么"确定"了——不是不知道自己想要什么，是对"想要"的力度减弱了。

### 通道三：契约深度 → 修为加速

**方向**：契灵资源 → 修仙资源

**操作**：利用契约深度带来的织上影响力，临时加速灵气吸收或灵根成长。俗称"用契灵给修炼插队"。

**机制**：
- 在修炼时主动调用契约能力——不是让契灵做什么，是利用契约锚点作为"额外灵根"
- 锚点在织上的存在可以充当"灵气收集器"——比单纯依靠灵根更高效
- 修炼速度临时提升 20%-50% [待测试]

**兑换率**：这本质上是"借用"而非"转化"——不是用契约深度换修为，是用契约深度的暂时消耗加速修为获取。

**限制**：
- 每次加速修炼后，契约深度下降约 5%-10%（锚点在修炼中被"磨损"）
- 下降的深度需要通过重新确认仪式或支付额外代价来恢复
- 长期使用此方法会导致契约深度持续保持在低水平——"浅契约"状态下能力输出下降，且更易反噬
- 每次加速修炼后，契灵会对修炼者产生"额外关注"。不是好感或恶意——是契灵在修炼者的织上留下了更多"嗅觉痕迹"，使契灵更容易感知到修炼者的状态。对某些修炼者来说，这意味着隐私下降。

**叙事含义**：让契灵"掺和"到自己的修炼中。修炼者会感觉到修炼时有一种"被看着"的感觉——不是镜那种记录式的看，是一种存在性的关注。有些修炼者因此无法专注于修炼（适得其反）。

|:---|:-----|:------:|:---------|:-----|

## 四、债务螺旋预防规则

### 什么是跨体系债务螺旋

跨体系债务螺旋是指：用一个体系的资源去修复另一个体系的问题，然后用后者的资源去修复前者的问题——形成一个循环依赖，总损耗不断积累，最终两者同时崩塌。

### 已知螺旋模式

**模式一：修炼-代价循环**

1. 契主道心不足 → 锚点不稳定
2. 用修为经验加固锚点（通道一，损耗 30%）
3. 修为损失后修炼效率下降 → 用契约深度加速修炼（通道三，深度消耗 5-10%）
4. 契约深度下降使锚点更不稳定 → 需要更多道心或修为来加固
5. 回到步骤 1——但每次循环都有损耗

完成 3 次循环后：总修为消耗约初始的 150%（其中约 50% 被织吸收），契约深度降至原来的 60%。

**模式二：代价-灵根循环**

1. 契约的代价池接近沸腾 → 用灵根经验降温（灵根经验转为代价池缓冲）
2. 灵根经验损失 → 灵根活跃度下降 → 修炼效率降低
3. 修炼效率降低 → 无法快速积累修为来维持道心 → 道心衰退
4. 道心衰退 → 锚点更不稳定 → 代价池升温加速
5. 回到步骤 1——但每次冷却代价池需要的灵根经验越来越多

### 预防规则

1. **三转换原则**：任何一个转换通道的使用频率不超过每季度 1 次。如果需要更频繁地使用某通道，说明该双修者的体系平衡已经出现问题，应优先调整基础结构而非依赖转换来维持。
2. **不跨通道循环**：如果一个转换通道已经被使用，在同一个自然月内不要使用与之形成循环的另一个通道。
3. **止损线**：当三周期内总转换损耗已经超过初始资源储备的 30% 时，立即停止所有跨体系转换。进入"体系隔离"状态至少一个月——只在一个体系内运作，不进行任何跨体系操作。
4. **优先解除契约而非转换救急**：当面临代价池沸腾时，用转换来降温是最差的选择。更好的选择是：解除一份不必要的契约、重新谈判代价条款、或请资深契主进行锚点维护。转换应该是最后的手段。

## 五、镜的跨体系转换数据

> 镜追踪了 214 例有记录的跨体系转换行为。
>
> - 短期成功（解决当前问题，6 个月内未见副作用）：31%
> - 中期代价出现（解决了一个问题但创造了另一个问题，6 个月至 2 年内）：45%
> - 长期失败（转换导致的链条反应超过最初问题的损害）：19%
> - 数据不足（失联、死亡、无法追踪）：5%
>
> 镜不评价这些数据是好是坏。镜只是记录。
>
> 但镜注意到一个模式：**那些用转换来"救急"而非"优化"的人，长期失败率是那些用转换来"战略规划"的人的 3 倍。**
>
> 换句话说：用转换来堵漏洞的，漏洞会在别处重新打开。用转换来重组结构的，结构更可能保持稳定。

## 六、关于转换的常见误解

### "转换可以替代修炼/履约"

不能。转换可以辅助，但不能替代。一个用修为换契约深度的人，仍然需要定期确认锚点、支付代价。一个用契约深度加速修炼的人，仍然需要每天完成修炼动作。转换只是提供了"微调"的杠杆，不是"跳过"的捷径。

### "多用几次转换就能找到最优规律"

织的兑换率不是固定的常数——它受到织局部密度、当前季节（某些季节织的流动性更高）、甚至附近大型事件的影响。同一个转换通道在不同时间、不同地点的兑换率可能差异高达 15%。无法通过反复尝试来"优化"出一个稳定汇率。

### "转换上的损耗是浪费"

损耗不是浪费。被织吸收的能量不会消失——它只是回到了本源，变成了织的普遍密度的一部分。你的转换行为本身在维持织的流动性。从某种角度看，每一笔转换都是对织的一次微小的"灌溉"。
