# 跨体系联动 · 兼容性规则

> 谁能走两条路？代价是什么？极限在哪里？
> 本文件给出硬边界——不模糊、不妥协、不可绕过。

---

## 一、兼容性矩阵

三类存在者 x 两套体系 = 六种组合。以下矩阵总结了每种组合的兼容性状态：

| 存在者类型 | 修仙 | 契灵 | 综合评述 |
|:----------|:----|:-----|:---------|
| **凡人** | 可开始（需有灵根） | 可开始（不需要灵根） | 凡人可以在两者之间选择——但通常建议只选一条 |
| **纯修仙者** | 完全 | 有限（见下文） | 契约可以签，但有道心损耗和突破阻碍 |
| **纯契主** | 有限（见下文） | 完全 | 可以修炼，但灵根"被污染"，上限低 |
| **双修者** | 限制（上限化神） | 限制（上限2契约） | 两条路都走，两条路都走不远 |
| **散功者** | 可重新开始（道心保留） | 可开始（无额外障碍） | 失去修为的修仙者转向契灵术是已知路径 |
| **破契者** | 极难（道心已被碎片化） | 危险（契约体系濒临崩塌） | 破契者尝试修仙的动机通常是绝望而非选择 |

## 二、纯修仙者可以签契约吗？

**可以。但每次签约都是有代价的——且代价超出契灵本身索要的部分。**

### 规则一：道心容量缩减

修仙者的道心不仅支撑修炼，还支撑其在织上的存在一致性。每签订一份契约，道心都会因为"需要同时维持两条织上路径"而承受额外消耗。

- 第一份契约：道心最大容量永久减少 15% [待测试]
- 第二份契约：道心最大容量再减少 20%（累计 35%）
- 第三份契约：道心最大容量再减少 25%（累计 60%）

这意味着：一个金丹期修仙者（基础道心容量 100）签一份契约后，他的有效道心容量降为 85。这不会立刻造成问题——但如果他原本的道心利用率就高（>80%），缩减后可能直接触发道心不稳。

**叙事含义**：修仙者每签一份契约，就在自己的道心上刻了一道裂痕。裂痕不会愈合。

### 规则二：道心与契灵必须哲学兼容

契约能否安全签订，取决于修仙者的道心方向（[道心本质](/02.修仙体系扩展/07_道心与劫.md)）与契灵的本质是否兼容。

| 道心倾向 | 兼容的契灵类型 | 不兼容的契灵类型 |
|:--------|:-------------|:---------------|
| 力行（以行动证道） | 残响（继承斗争意志）、地缚灵（扎根） | 概念体（过于抽象）、镜中我（过于内省） |
| 求真（以知识证道） | 概念体、镜中我 | 地缚灵（区域限制与求知冲突） |
| 自然（以无为证道） | 概念体（沉默/遗忘）、残响（随缘） | 天道碎片（使用即干涉，与无为冲突） |
| 兼爱（以关系证道） | 地缚灵（守护）、残响（传承） | 镜中我（过度自我关注） |

**如果修仙者签订了一份与道心不兼容的契约，在签约后的 30 天冷静期内，反噬概率提升 300%** [待测试]。冷静期后，如果契约维持，反噬概率逐渐回落，但永久高于普通契主约 50%。

### 规则三：突破阻碍

拥有活跃契约的修仙者在突破境界时，需要克服额外的阻抗：

- 每份活跃契约使突破时的天劫强度增加约 20% [待测试]
- 天劫类型可能变异——不仅测试修炼者的存在一致性，还测试其"在外部契约影响下是否仍能保持自我"
- 突破失败后，契约锚点可能在天劫回响中松动，需要额外的锚点加固仪式

这意味着修仙者签了契约后，突破不再只是"自己的事"——契灵也会参与其中，有时作为负担，有时——极少情况下——作为意外的助力。

### 案例

一个名为何守拙的金丹期散修，为了获得一项关键的情报能力（在绝灵之地感知灵气波动），与一个残响签订了契约。残响生前是一位元婴期的感知专精修士。

何守拙签了契约，获得了感知能力。代价是他从此无法进行长时间闭关——契约锚点在深度入定中会松动。他的元婴期突破尝试失败了两次，不是因为境界不够，是因为每次闭关到第七天，契约就会"呼唤"他——不是契灵主动叫他，是锚点在深度入定中像一个松动的地桩，在风中摇晃。

他最终放弃了元婴期的追求。保留了契约。他说："我选了知道，而不是更高。"

## 三、纯契主可以修仙吗？

**可以。但灵根"不干净"——修炼效率低，上限低。**

### 规则一：灵根污染

签订契约的行为会在修仙者的灵根上留下"外部痕迹"。[不同类型的契灵](/03.全新异能体系/03_契约类型.md) 在灵根上留下不同类型的痕迹：

| 契灵类型 | 对灵根的污染 | 表现 |
|:--------|:------------|:-----|
| 残响 | 灵根上叠加了不属于自己的修炼记忆 | 修炼时偶尔"用错方法"——残响的习惯覆盖了自己的习惯 |
| 地缚灵 | 灵根与特定地理位置绑定 | 离开绑定地后修炼效率下降 40%+ |
| 概念体 | 灵根被抽象化——对灵气的感知变得"概念化" | 可以感知灵气但难以吸收——就像能看到水但喝不到 |
| 镜中我 | 灵根与镜像层重叠 | 修炼时常常不确定"是自己在吸收灵气，还是镜中我在吸收" |
| 天道碎片 | 灵根上出现"法则级异常" | 修炼效果不稳定——有时暴涨，有时暴跌，不可预测 |

### 规则二：起步惩罚

一个有契约在身的凡人开始修仙时：

1. **修炼效率降低**：初始修炼效率约为纯修炼者的 60% [待测试]。这不是因为契主"没天赋"——是因为灵根已经部分被契约占用。
2. **首次突破惩罚**：从凡人到炼气的第一次顿悟突破概率从 90% 降至约 50% [待测试]。顿悟要求"四道合一的第一瞥"——但契主的意识已经被契约分割，较难体验完整的合一感。
3. **功法建设困难**：功法需要连续修炼多天才能建成。契主同时需要维护契约锚点，连续修炼的难度高于纯修仙者（断签率高约 2-3 倍）。

### 规则三：上限锁定

**契主修仙永远无法达到最高三个境界。**

| 境界 | 纯契主能否达到 | 原因 |
|:----|:-------------|:-----|
| 炼气 | 可以 | 基础入门的门槛不涉及存在完整性 |
| 筑基 | 可以 | 需要额外努力但可行 |
| 金丹 | 可以（困难） | 心劫要求面对真实自我——契主的自我已经因契约而碎片化 |
| 元婴 | 极难 | 言劫要求语言与存在一致——契约影响了存在的一致性 |
| 化神 | 理论可能（无记载案例） | 综合劫需要多重自我统一——契主的多重契约使统一极困难 |
| 渡劫 | 不可能 | 存在形态的根本转化需要纯粹的自我——契主已经没有 |
| 大乘 | 不可能 | 飞升的前提是自我与天道完全对齐——契约锚点是对齐的障碍 |

**底层原因**：[法则五（无体系可兼得广度与深度）](/01.系统/世界观底层规则.md#法则五无体系可兼得广度与深度) 决定了契主选择了广度（多契约多样性），就自动放弃了深度（修炼到顶的可能性）。

## 四、双修者 Dual Practitioner（双修者）

同时追求两条路的人——在 [04.跨体系联动/03_双修路径.md](/04.跨体系联动/03_双修路径.md) 中有详细路径分析，此处只列硬边界。

### 数量限制

| 维度 | 纯修仙者 | 纯契主 | 双修者 |
|:----|:--------|:------|:-------|
| 最大安全契约数 | 1（修仙为主） | 4（安全上限） | **2** |
| 最大契约数（含不稳定） | 3 | 5+（破契者） | **3** |
| 最高修仙境界 | 大乘 | 化神（理论） | **化神** |
| 道心最大容量保留 | 100% | 60%（基础） | **70%** [待测试] |

### 平衡比例

双修者必须在"契约影响力"和"修炼进度"之间维持一个动态平衡。如果契约影响力超过修炼进度太多，触发的是**双重反噬**——道心崩塌与锚点断裂同时发生。

契约影响力 vs 修炼进度的比值：

```
R = (契约数 × 平均契约深度) / (境界系数 × 灵根总值)
```

- 安全范围：R < 0.6 [待测试]
- 警告范围：0.6 <= R < 0.8
- 危险范围：R >= 0.8

**当 R >= 0.8 时，双修者必须在 7 天内采取行动**——解除一份契约、降低契约深度、或紧急提升修炼进度。否则触发双重反噬。

### 双重反噬

双修者最危险的状态。当契约反噬和修仙心魔/走火入魔同时发生时：

1. 反噬动摇锚点 → 锚点不稳定在织上造成局部扰动 → 扰动干扰灵根 → 灵根失控触发灵气暴走 → 暴走进一步撕裂锚点
2. 心魔削弱道心 → 道心不稳降低契约维持能力 → 契约松动加速反噬 → 反噬加深心魔

两个循环相互加速，通常在数小时内达到不可逆状态。

**已知的应对方式**（按生存率排序）：
1. 立刻解除所有契约（承受重度反噬）→ 然后专心疗伤 —— 生存率约 40%
2. 主动散功（修为归零）→ 保留契约 —— 生存率约 30%
3. 求助高阶修士或资深契主进行"剥离仪式" —— 生存率约 50%（但需要两个体系的高手同时在场，极难安排）
4. 什么都不做 —— 生存率低于 5%

## 五、镜的观察能力

[镜](/02.修仙体系扩展/04_镜之起源.md) 可以观测两套体系中的存在，但不能被契约。

**为什么镜不能被契约？**

1. **结构原因**：镜是织上的"结构"而非"存在"。它没有可建立锚点的意识核心。试图与镜签订契约，就像试图与一扇门握手——门没有手。
2. **功能原因**：镜的存在功能是记录，不是交互。如果镜可以被契约，它的记录中立性会受到根本质疑——"镜的记录是否受到其契约关系的影响？"即使镜本身不受影响，外界的质疑也会破坏其记录的公信力。
3. **法则原因**：镜是 [法则三（观测影响现实）](/01.系统/世界观底层规则.md#法则三观测影响现实) 的产物。如果镜可以参与契约（法则三的应用之一），会造成法则三的自我指涉悖论——观测者被观测对象影响，观测结果的可靠性降为零。

## 六、兼容性决策树

一个角色决定是否兼容两种体系，可以按以下决策树推理：

```
开始：我是否已经在一个体系中有投入？
  ├── 是（修仙）→ 我想签契约吗？
  │   ├── 是 → 道心与契灵兼容吗？→ 否 → 强烈不建议
  │   │                                   → 是 → 接受15%道心容量缩减 → 可签1份
  │   └── 否 → 保持纯修仙 → 安全
  │
  ├── 是（契灵）→ 我想修仙吗？
  │   ├── 是 → 接受60%修炼效率 + 上限锁定 → 可尝试
  │   └── 否 → 保持纯契主 → 安全
  │
  └── 否（凡人）→ 我选哪条路？
      ├── 修仙 → 需要灵根 + 长期主义
      ├── 契灵 → 不需要灵根 + 短期见效 + 长期代价
      └── 双修 → 同时接受两套限制 + 存活率约40% → 非常不建议
```
