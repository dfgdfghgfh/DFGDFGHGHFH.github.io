# 修仙体系扩展 · 概述

> 修仙养成工具中埋下了世界观的种子——境界、灵根、道心、功法。本文件集将其发芽为完整的叙事土壤。不修改已有机制，只揭示其背后的世界。

---

## 一、本扩展集的范围

**02.修仙体系扩展** 是对修仙养成工具中已存在的数据结构和叙事暗示的深度展开。它不做三件事：

1. **不修改游戏机制**——config.json、training.json、techniques_full.json 等数据文件的数值和规则保持不变。本扩展集只解释"为什么"，不改动"是什么"。
2. **不替代现有设定**——零之书、道藏阁、秘籍数据等已有内容在本扩展集中被引用和深化，而非覆盖。
3. **不创造与 [01.系统/世界观底层规则.md](/01.系统/世界观底层规则.md) 矛盾的内容**——底层五法则的推导结果。如果有扩展内容触及法则边界，以底层规则为准。

本扩展集会做三件事：

1. **揭示**——已有机制背后的叙事逻辑。为什么突破有概率？为什么溢出是边际递减？为什么连签有乘数效应？每个数值背后都有一个世界层面的原因。
2. **填充**——世界中存在但未展开的部分。历史、神话、宗门、人物。[01.系统/术语对照表.md](/01.系统/术语对照表.md) 中定义的术语在这里获得血肉。
3. **预留**——未解之谜和叙事钩子。不是所有问题都需要答案。10_未解之谜.md 中列出的开放问题是有意为之的创作空间。

## 四、阅读路径建议

本扩展集设计为非线性的参考文档，但也支持顺序阅读：

- **对世界观整体感兴趣者**：01 → 05 → 02 → 03 → 04 → 06 → 07 → 08 → 09 → 10
- **对机制背后的哲学感兴趣者**：05 → 07 → 02 → 03
- **对叙事声音感兴趣者**：04 → 01（镜的视角下） → 06
- **寻找创作灵感者**：10 → 09 → 08（从开放问题开始）

## 五、版本说明

- 本扩展集为 v1.0，与修仙养成工具 v1.0.0 的数据结构对齐。
- 如果修仙养成工具的 config.json（境界阈值、突破率、灵根等）发生变更，本扩展集的相应描述需要同步更新。
- 如果 [01.系统/跨体系联动总纲.md](/01.系统/跨体系联动总纲.md) 中新增交互模式，08_宗门与势力.md 中的宗门立场可能需要补充。
