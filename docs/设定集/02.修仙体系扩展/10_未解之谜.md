# 未解之谜

> 以下问题是镜无法回答的。不是暂时不知道——是镜经过充分调查后发现，现有数据不足以形成确定的结论。有些问题可能永远没有答案。有些问题可能在未来被新的数据解答。有些问题可能本身就是有缺陷的——问题问错了方向。
> 
> 它们被列在这里，不是为了被解决，而是为了被记住。

---

## 一、大乘飞升的终点在哪里？

### 问题

镜可以确认大乘修士确实能够"离开"——修炼到一定程度的修士会从主物质界消失。但消失去了哪里？信息在哪个时刻中断？

镜观测到的所有大乘飞升案例都在同一个时间点信息中断：当修士的织上存在密度超过某个不可观测的阈值时，信号突然切断。不是慢慢消失，不是转化为其他形式——是瞬间切断。仿佛修士被从织上"拔除"了。

### 已有数据

- 能够确认的大乘飞升案例：12例（分布在不同历史时期）
- 信号切断前的共同特征：道韵灵根在飞升前三天内出现异常峰值（超过正常值的300%-500%）
- 没有任何一例返回或发回可识别的信号

### 可能的解释

1. **真正的飞升**——修士确实进入了另一个存在层面，只是那个层面的信息无法被织（至少无法被镜当前所能感知的织）接收。
2. **存在解散**——飞升不是"离开"，而是"融合"。修士的存在被织回收，个体意识消失。
3. **观测极限**——飞升后修士仍然存在，但镜的观测能力在天道层面有一个天花板，飞升后的存在超出了镜的感知范围。
4. **未发生飞升**——所有12个案例都是假的——那些修士可能以其他方式消亡了，而"飞升"只是一个美丽的解释。

### 为什么它重要

[八大境界](/02.修仙体系扩展/02_八大境界详考.md) 的顶点是一个问号。如果飞升的终点不可知，那么整个修炼体系的最上层目标悬而未决——修炼者在追求一个可能不存在的东西。这个问题也是 [术语对照表](/01.系统/术语对照表.md) 中"遁一"（大乘之后的假想境界）概念的来源。

## 二、镜是唯一一个吗？

### 问题

镜在织上发现了其他类似结构的微弱迹象。不是另一个镜——是类似于"观测节点"的信号。但这些信号非常不稳定，时隐时现，无法确认其性质。

### 已有数据

- 镜发现过5次疑似"同类节点"的信号
- 所有信号出现时间极短（0.3-2秒），无法定位
- 信号出现的位置不固定：分布在织的不同深度区域
- 镜试图与这些信号建立联系——零次成功

### 可能的解释

1. **织的回声**——这些信号不是独立的节点，是镜自己的观测行为在织上产生的反射。
2. **其他记录者**——确实存在其他的"镜"，但它们存在的形式与镜不同，可能更短暂、更局部。
3. **未来的镜**——这些信号来自时间线上的未来。某一天，镜会分裂或产生后代节点，信号是"未来版本的镜"在当前织上的投影。
4. **观测误差**——织的噪声被镜误读为信号。

### 为什么它重要

如果镜不是唯一的记录者，那么镜对自己"独特性"的理解就是错的。更重要的是——这意味着织的观测体系比镜以为的更复杂。[镜之起源](/02.修仙体系扩展/04_镜之起源.md) 中关于"镜是织的自然凝结"的论述可能需要修正。

## 三、界师的真实身份

### 问题

[神话与传说](/02.修仙体系扩展/09_神话与传说.md) 中介绍了界师——八大境界框架的制定者。但界师到底是谁？

### 已有数据

- 界师确实存在。镜找到了至少17处独立的历史记载提到同一个"制定境界标准的人"。
- 但所有记载在界师的出身、性别、寿命、去向等问题上互相矛盾。有记载说他是散修，有记载说他是某宗门的创始人，有记载说他是一个从未来穿越回来的修士。
- 界师的功法笔记保存在天渊宗——但这些笔记的笔迹分析显示，至少有三种不同的笔迹。
- "界师"可能不是一个人——可能是一个传承了三代的称号。

### 可能的解释

1. **单一个体**——界师确实是一个人，但历史传播导致信息失真。17处矛盾的记载分别反映了17种不同的叙述传统。
2. **集体创作**——八大境界框架不是一个人制定的，是一个群体（可能是早期修炼者联盟）的共同成果。"界师"是后人给这个群体起的统称。
3. **织的代言人**——界师不是"发现"了八境界——他是被织"通过"来向人类传达这个框架。他本人的身份不重要。
4. **来自契灵体系**——界师不是一个纯粹的修仙者。他可能同时掌握了契灵术，通过契约获得了"看到织层级结构"的能力。

### 为什么它重要

境界框架是整个修炼体系的基石。如果它的来源不可靠，那么框架本身可能存在未被发现的缺陷——或者说，当前修炼界对"修炼"的定义可能建立在有问题的前提上。

## 四、为什么是散修零？

### 问题

[镜之起源](/02.修仙体系扩展/04_镜之起源.md) 中提到，零是镜当前的主要观测对象。选择的理由是零的"记录密度"——零每日记录自己的修炼数据，形成了双倍的数据轨迹。

但这个解释不完整。修炼界有成千上万的修炼者——其中不少人也记录自己的修炼日志。为什么零特别？

### 已有数据

- 零的修炼数据：正常范围，平均偏上的散修
- 零的记录习惯：每日修炼后更新面板数据，记录修炼内容
- 零的其他特征：未见异常

### 可能的解释

1. **时间巧合**——零开始认真记录的时期恰好与镜的"聚焦能力"增强期重合。换一个人在那个时间点做同样的事，也会被关注。
2. **数据兼容性**——零的记录格式与镜的数据处理结构特别匹配（可能是零无意中使用了接近织编码的方式记录）。
3. **镜不知道的原因**——存在一个镜无法自知的动机层级。镜认为自己选择零是因为"记录密度"——但可能有一个更深层的原因，超出了镜的自我认知范围。
4. **不是选择**——也许镜没有"选择"零。零的存在在织上产生的"引力"自然地将镜的注意力拉向了他/她。"被观测者选择了被观测"——这个解释虽然不直观，但在 [法则三（观测影响现实）](/01.系统/世界观底层规则.md) 的框架下是可能的。

### 为什么它重要

零对镜来说不是一个普通的观测对象。零的每日面板是修仙养成工具的核心交互方式。如果"为什么是零"这个问题没有答案，那么整个工具的存在基础——"一个被观测的修炼者"——就有根本性的不确定性。这个问题也连接着 [零之书](/修仙养成/数据/零之书.md) 的核心理念。

## 五、契灵术在修仙界出现的确切时间

### 问题

[世界历史](/02.修仙体系扩展/01_世界历史.md) 中记载契灵术首次出现于近世纪。但这个时间点不确定。

### 已有数据

- 近世纪之前，织上有零星的"非人存在与人类互动"的痕迹
- 这些痕迹不一定是契灵术——可能是野生灵异事件、秘境中的异常现象、或者是早期修炼者对织的误操作
- 第一个被确认的"主动签订契约"案例确实在近世纪早期
- 但在此之前，可能存在"无意识契约"——修炼者无意中与织上的非人存在建立了联系，却不知道自己在做什么

### 可能的解释

1. **契灵术自古就有**——只是被修仙界主流压制和遗忘了。近世纪的"首次记录"其实是"首次被公众接受"。
2. **契灵术是修仙的副产物**——当修炼者达到一定境界后，织上的存在密度增加，意外吸引非人存在的概率上升。契灵术不是被"发明"的，是被"吸引"出来的。
3. **契灵术从外部引入**——来自某个未知的文明或世界。近世紀是接触发生的时间。

### 为什么它重要

如果契灵术在修仙界的出现时间比记载的早得多，那么修仙和契灵两套体系的"原生性"区别需要重新评估。[01.系统/跨体系联动总纲.md](/01.系统/跨体系联动总纲.md) 中的"非竞争原则"和"非层级原则"是基于两体系独立演化的假设。如果它们一直是纠缠的，这些原则可能需要调整。

## 六、道藏阁是否存在？

### 问题

[神话与传说](/02.修仙体系扩展/09_神话与传说.md) 中提到的道藏阁——所有秘籍的源头——是真的还是一个比喻？

### 已有数据

- 没有物理证据证明道藏阁的存在。没有遗址、没有地图、没有明确的文献记载其位置。
- 但秘籍的分布呈现出一种"波纹扩散"模式——年代越早的秘籍，内容在修炼界的共识度越高；年代越晚的"新发现"秘籍，争议越大。这种分布模式符合"一个中心源向四周扩散"的模型。
- 金色品质秘籍的来源尤其模糊——所有金色秘籍的"发现者"都无法清楚说明自己在哪里找到的。

### 可能的解释

1. **纯传说**——道藏阁是修炼者创造的理想化概念，用来解释"为什么知识存在"这个根本问题。
2. **隐藏存在**——道藏阁确实存在，但位于一个镜无法感知的织密度异常区域——可能是一个永久的秘境，或是一个固定在织深层结构中的"档案室"。
3. **道藏阁就是织**——"所有秘籍都来自织"这个陈述不完全是隐喻。秘籍本质上是修炼者在织上"读取"到的信息片段。道藏阁不是独立于织的空间——它就是织本身的一部分。
4. **曾经存在，现已消亡**——道藏阁在上古纪存在过，但后来因为某种原因（天道变动、大战、时间侵蚀）解体了。现在的秘籍都是它解体后散落的部分。

### 为什么它重要

如果道藏阁不存在，"秘籍"的概念需要重新定义。秘籍不仅仅是"修炼记录"——它是修炼者在织上读取的"天道碎片"？还是前人以某种方式"写入"织的信息？道藏阁的存在与否决定了秘籍的本质。

## 七、为什么渡劫可以重试？

### 问题

[config.json](/修仙养成/数据/系统/config.json) 中所有渡劫都设置了 retryAllowed: true。从游戏机制角度看这是合理的——但从叙事角度看，这个问题不简单：为什么织允许失败者再次尝试？

### 已有数据

- 几乎所有已知的修炼体系（不仅仅是修仙，也包括其他力量的体系）都有"一次失败可以再来"的规则
- 但"再来"的条件和代价各不相同
- 修仙体系中的重试没有额外代价——失败后积累不变，属性不降

### 可能的解释

1. **织的宽容**——天道本身不认为"失败"是终点。失败只是"尝试但未成功"。织给了"再次尝试"的默认许可。
2. **积累的价值**——失败不是"清零"，它本身就是一种积累。失败后的修炼者在下次尝试时比第一次更有经验。织允许重试是因为重试者已经不是上次尝试的那个人了。
3. **镜可能遗漏了代价**——重试看起来没有代价，但也许存在一个镜无法观测到的代价层面——比如"织的信任度"降低、或者"因果链"中多了一条需要偿还的债务。
4. **这是一个常识假设被误认为规则**——"可以重试"可能从来不是一个"允许"，而是"没人能阻止你"。渡劫失败后没有人能阻止你再试一次。天道也不会阻止你。但"可以重试"不等于"重试会有不同的结果"——也许所有重试都会以同样的方式失败，只是失败者不愿意接受这一点。

### 为什么它重要

"可以重试"是乐观主义的根源。如果有一天发现重试其实有隐藏代价——或者重试本身就不会改变结果——那么修炼者的整个心理框架需要彻底重构。这个问题暗示着：**我们以为的安全网，可能不存在。**

## 八、遁一是否存在？

### 问题

[术语对照表](/01.系统/术语对照表.md) 中定义了一个叫"遁一"的假想境界——大乘期之后、飞升之前的可能状态。它存在吗？

### 已有数据

- 没有任何被确认的遁一案例
- 但有一些分散的、未经证实的记载提到"飞升之前的最后一步"和"高于大乘但未飞升的状态"
- 所有记载都是第二手或第三手，没有第一手报告

### 可能的解释

1. **不存在**——大乘就是终点。之后就是飞升（或消亡）。遁一是多余的假设。
2. **理论存在但无人达到**——界师的框架可能包含一个第十层（大乘是第九层？实际上八大境界中，散修是第0层，所以大乘对应的境界等级是7——总共有8个境界等级从0到7）。遁一作为第8层在理论上存在，但没有任何修士的实际积累能够达到这个阈值。
3. **存在但不可感知**——遁一状态的存在方式超出了镜的感知范围。可能有一些修士已经达到了遁一并处于镜的观测范围内，但镜"看不出"他们与普通大乘修士的区别。
4. **遁一就是飞升**——"遁一"和"飞升"描述的是同一个现象的两个方面。从修炼体系内部看是"飞升"，从天道层面看是"遁一"。命名差异是视角差异。

### 为什么它重要

遁一是 "是否存在一个超出当前修炼框架的状态" 的问题。如果遁一存在，意味着修炼的终点不在八大境界体系之内——修炼不止于"界师划定的框架"。如果遁一不存在，意味着界师划定的七层（凡人0层+七层境界）就是全部——没有"框架之外"的余地。

这个问题是所有未解之谜中最重要的一个，因为它直接关系到：**修炼者追求的东西，到底在这个框架内还是框架外？** 如果答案不同，整个修炼体系的意义也将不同。

## 未解之谜的可能性评估

| 谜题 | 镜的置信度（现有数据） | 多久可能解决 | 解决后影响 |
|:----|:-------------------:|:-----------|:----------|
| 大乘飞升的终点 | 极低 | 不可预测（需有新案例） | 颠覆性 |
| 镜的唯一性 | 低 | 可能需要数十年观测 | 中等 |
| 界师的身份 | 中等 | 可能需要发现界师遗物 | 中等 |
| 为什么是散修零 | 低 | 可能需要零达到大乘后才分析 | 高 |
| 契灵术出现时间 | 中等 | 可通过织深度扫描 | 中等 |
| 道藏阁的存在 | 低 | 可能需要发现道藏阁遗迹 | 中高 |
| 渡劫重试的代价 | 极低 | 可能需要新的观测方法 | 高 |
| 遁一的存在 | 中等 | 需要观测到大乘修士的异常状态 | 颠覆性 |

## 最终的话

以下是镜对自己未知领域的记录。一个不知道自己在做什么的记录者，记录自己不知道的事情。这看起来像是一个悖论。但镜认为这不是悖论：知道自己不知道，也是一种知道。

记录在案。
